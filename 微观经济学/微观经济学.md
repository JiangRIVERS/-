## 读书笔记
## 第一章 经济学核心概念
### A. 为何学习经济学
#### 稀缺与效率：经济学的双重主题
+ 经济学的定义
  1. 经济学研究的是一个社会如何利用稀缺的资源生产有价值的商品，并将它们在不同的个体之间进行分配
+ 稀缺与效率
  1. 物品和资源是稀缺的，以及社会必须有效利用这些资源
  2. 稀缺是指这样一种状态：相对于需求，物品总是有限的
  3. 效率是指最有效地利用社会资源以满足人类的愿望和需求
  4. 经济效率要求在给定技术和稀缺资源的条件下，生产最优质量和最多数量的商品和服务。在不会使他人境况变坏的前提下，如果一项经济活动不再有可能增进任何人的经济福利，则该经济活动就被认为是有效率的
  5. 经济学的精髓之一在于承认稀缺性是一种现实存在，并探究一个社会如何进行组织才能最有效地利用其资源
+ 微观经济学与宏观经济学
  1. 宏观经济学研究经济的总体运行
#### 经济学的逻辑
1. 经济生活是由一系列活动组成的复杂集合，包括购买、销售、讨价还价、投资、劝说等。经济科学的最终目的就是要理解这些复杂的活动
2. 科学方法：观察经济事件、利用统计分析和历史记录
3. 经济推理中常见的谬误：
  + 前因后果谬误：未必“前因后果”
  + 不能保持其他条件不变
  + 合成谬误：总体未必等于部分之和
4. 在考虑经济问题时，我们必须区分事实本身和它是否公平两个方面的问题。实证经济学讨论的是经济社会的事实，而规范经济学则涉及价值判断
#### 热切的心情，冷静的头脑
1. 经济科学的最终目的是改善人们的日常生活条件
2. 一个自由和有效的市场并不必然地使收入分配得到全社会的认可
### B. 经济组织的三个经济问题
三个最基本的经济问题：生产什么、如何生产、为谁生产（谁来想用经济活动的成果）
#### 市场经济、指令经济和混合经济
1. 通常我们区分两种本质不同的经济组织方式。一个极端是，政府制定大部分经济政策。另一个极端是，决策由市场来做出
2. 市场经济是一种主要由个人和私人企业决定生产和消费的经济制度
3. 指令经济是由政府制做出有关生产和分配的所有重大决策
4. 混合经济既带有市场经济的成分也带有指令经济成分
5. 当今绝大多数社会都采用混合经济制度
### C. 社会的技术可能性
#### 投入和产出
1. 投入指的是生产商品和服务的过程所使用的物品或劳务
2. 产出是指生产过程中创造的各种有用的物品或劳务
3. 投入的另一个名称叫生产要素，它们可以被划分为三大基本范畴：土地、劳动和资本
4. 资本是一个经济体为了生产其他的物品而生产出来的耐用品
5. 一个社会必须决定：（1）产出什么、产出多少（2）如何产出，即用何种技术和要素进行投入，以得到令人满意的产出（3）为谁生产，即如何分配这些产出
#### 生产可能性边界
1. 生产可能性边界（PPF），表示在技术知识和可投入品数量既定的条件下，一个经济体所能有效率地得到的最大产量
2. 对未来投资较多的国家将拥有更多的投资品和更多的消费品 
#### 机会成本
1. 由于资源是稀缺的，因此，我们必须不断地决定如何利用我们有限的时间和收入。你需要放弃一些事情--存在着若干可以被放弃的机会。这类有待被放弃的非最优的事情或东西，体现着你进行该项决策的机会成本
2. 机会成本是相应的所放弃的物品或劳务的价值
#### 效率
1. 总体经济效率的一个非常重要的方面就是生产的效率。效率意味着该经济处在PPF边界“上”，而非边界“内”
2. 当一个经济体无法在不减少一种产品产量的前提下生产出更多的另一种产品时，即当选择点处在生产可能性边界上的时候，我们就说此时是有效率的生产
3. 商业周期和环境恶化所产生的浪费
  + 经济危机是产生非效率的根源之一
  + 在经济周期的衰退阶段（由于宏观经济的刚性，并非技术性遗忘）也会出现非效率
  + 当市场无法真实反映事物的稀缺性时，例如环境恶化，另一种非效率就会发生
4. 旨在理解怎样才能将经济学强有力的思想用于人类社会核心问题的分析，这就是我们学习经济学最根本的理由
## 第一章附录
#### 边际值的斜率
1. “边际”是经济学中最重要的概念之一，通常的意思是”附加的“或者”额外的“。
2. 边际成本：当一家企业生产额外一单位产品时所额外付出的成本
## 第二章 现代混合经济
每个人都力图用好他的资本，使其产出能实现最大的价值。一般来说，他既不会企图增进公共福利，也不知道他能够增进多少。他所追求的仅仅是一己的安全或私利。但是，在他这样做的时候，有一只看不见的手在引导着他去帮助实现另一个目标，尽管该目标并非是他的本意。追逐个人利益的结果，使他经常地增进社会的利益，其效果要比他真的想要增进社会的利益时更好。--亚当斯密《国富论》（1776年）
1. 在一个良好运转的市场机制中，追求私人利益可以增进社会福利
2. 世界各国都是一种混合经济，即市场上的私人企业，以及监管、税收和规划等各政府部门的结合体
### A. 市场机制
1. 大部分经济活动都是私人企业经由市场机制而进行和完成的
2. 这些活动通过价格和市场机制而得到了协调
#### 不是混乱而是经济秩序
1. 市场应被理解成一种买者和卖者决定价格并交换物品、劳务或资产的机制
2. 市场的核心作用是决定商品的价格
3. 价格是物品的货币价值
4. 价格在市场机制中起着平衡的作用
5. 市场均衡代表了所有不同的买者和卖者之间的一种平衡
6. 在某一价格水平上，买者所愿意购买的数量正好等于卖者愿意出售的数量，这一价格就达成了供给和需求的均衡
#### 市场如何解决三大经济问题
1. 市场经济同时解决了生产什么、如何生产和为谁生产这三个问题
2. 生产什么商品和劳务取决于消费者在每天购买决策中的货币选票
3. 如何生产取决于不同生产者之间的竞争
4. 为谁生产主要取决于生产要素市场的供给与需求
#### 二元君主
1. 经济的核心控制者是偏好和技术
2. 利润在奖励或惩罚企业并引导市场机制
#### 看不见的手
1. 在一个良好运转的市场机制中，追求私人利益可以增进公共福利
2. 在一定的条件下，完全竞争是有效率的
3. 市场不灵的一种情况是垄断以及其他形式的不完全竞争
4. 第二种不灵表现为市场的外溢效果或外部性：正面的外部性包括科学发现，而负面的外溢效果应包括环境污染
5. 对市场的最后一种指责是：其收入分配的后果在政治上或道义上是无法接受的
6. 在完全竞争和不存在市场不灵的情况下，市场会用其资源尽可能多地生产出有用的物品与劳务。但在存在垄断、污染或其他市场不灵的情况下，看不见的手的效率特征就可能会遭到破坏
7. 消费是所有生产的唯一的终点和目的
### B. 贸易、货币和资本
现代经济最显著的特征：贸易网络、货币、资本运用
#### 贸易、专业化和劳动分工
1. 专业化是指让个人或国家各自集中精力去完成某一种（或一系列）任务
2. 贸易生财的思想是经济学的核心见解之一
3. 专业化和贸易是提高生活水平的关键
#### 货币：交换的润滑剂
1. 货币还是一种能够便利交换的润滑剂
2. 货币供应适度是各国宏观经济政策的主要任务之一
#### 资本
1. 资本与土地和劳务不同，它在使用之前必须首先被生产出来
2. 资本的利用涉及到时间消费和迂回（例如：首先造鱼竿再钓鱼，而不是直接钓鱼）的生产手段
3. 个人占有资本并能从中获取利润，是资本主义的实质之所在
4. 产权指的是个人或企业所拥有的购买、出售和使用资本品和市场经济中其他财产的权利
### C. 政府--看得见的手
1. 政府的活动领域：公益事业、监管、补贴、征税...
2. 政府对于市场经济所行使的职能：
  + 政府通过促进竞争、控制诸如污染这类外部性问题，以及提倡公共品等活动来提高经济效率
  + 政府通过财政税收和预算支出等手段，有倾斜地向某些团体进行收入再分配，从而促进公平
  + 政府通过财政政策和货币政策附近宏观经济的稳定和增长，在鼓励经济增长的同时，减少失业和降低通货膨胀
#### 效率
1. 完全竞争指的是，没有一家企业或一位消费者足以影响整个市场的价格
2. 很多情况下，市场的竞争却不完全
3. 一个行动可能在市场交易之外有助于或有损于其他人的利益，存在着根本不发生经济支付的经济交易（外部性）
4. 外部性（或溢出效应）指的是企业或个人向市场之外的其他人所强加的成本或效益
#### 公共品
1. 公共品的两个特性：非相克性、非互斥性
2. 公共品是指这样一类商品，将该商品的效用拓展于他人的成本为零；无法排除其他人参与共享
3. 税收听起来好像是另一种“价格”。在此，它是我们为公共品所支付的价格
4. 税收并不是自愿支付的
#### 公平
1. 物品追随的是货币选票而不是最大满足
#### 宏观经济的增长与稳定
1. 财政政策就是税收权利和预算支出权利
2. 货币政策涉及到货币供应和利率水平
## 第三章 供给和需求的基本原理
本章的任务就在于描述在一个市场经济体系中生产流程如何展开的
### A. 需求表
#### 需求曲线
1. 需求向下倾斜规律几乎适用于一切商品，包括玉米片、汽油、大学教育和非法毒品等
2. 需求向下倾斜规律：当一种商品的价格上升时（同时保持其他条件不变），购买者便会趋向于购买更少的数量。同理，当价格下降且其他条件不变时，需求量会增加
3. 需求量趋于下降的原因：替代效应、收入效应
4. 需求曲线取决于市场需求
#### 市场需求
1. 市场需求曲线是将在每一价格水平下所有个人的需求量加总而得到的
#### 需求曲线背后的因素
| 影响需求曲线的因素 | 以汽车为例 |
| --- | --- |
| 1. 平均收入 | 收入增加，人们购买更多的汽车 |
| 2. 人口 | 人口增加导致汽车购买量增加 |
| 3. 相关商品的价格 | 汽油价格下降导致汽车需求增加 |
| 4. 偏好 | 拥有车成为身份的象征 |
| 5. 特殊影响 | 包括交通工具替代，汽车安全性，未来价格预期等 |
#### 需求的移动
1. 当商品价格之外的因素变化引起购买数量发生变化时，我们称这种变化为需求变动
### B. 供给表
一种商品的供给表或供给曲线体现的是：在其他条件不变的情况下，该商品的市场价格与生产者愿意生产和出售的数量之间的关系
#### 供给曲线
1. 供给曲线向上倾斜的重要原因之一是“边际收益递减规律”
#### 供给曲线背后的因素
1. 生产成本主要取决于投入品价格和技术进步
2. 
| 影响供给曲线的因素 | 以汽车为例 |
| --- | --- |
| 1. 技术 | 计算机化生产降低了生产成本，增加了供给 |
| 2. 投入品价格 | 工资下降，导致生产成本下降，从而供给增加 |
| 3. 相关商品的价格 | 如果卡车的价格下降，则轿车供给会增加 |
| 4. 政府政策 | 取消进口汽车配额和降低关税，会增加汽车总供给 |
| 5. 特殊因素 | 互联网购物和经济使消费者能以更低量的成本比较经销商的销售价，迫使高成本卖家出局 |
3. 只有在影响供给的其他因素发生变化时，供给才会发生变动，供给曲线也会因此移动
### C. 供给和需求的均衡
1. 供给和需求的力量会相互作用，产生均衡的价格和均衡的数量，即市场均衡。市场均衡发生在供给和需求达到平衡的价格和数量的点上
2. 在需求量和供给量相等的价格水平上，就会实现市场均衡。在均衡点上，价格既没有上升的趋势，也没有下降的趋势。我们也称均衡价格为市场出清价格。这意味着所有供给和需求的订单都已完成，账面上已经出清，需求者和供给者都得到了满足
#### 解释价格和数量的变动
1. 在经济学中，均衡意味着市场运作中不同力量处于平衡，从而价格和数量使购买者和供应商的愿望达成一致
2. 只有在这一水平上，买者和买者才会同时达到满足。也只有在这一价格水平上，价格和数量才不会继续有变动的趋势
3. 供给的增长是随着需求的增长而来的
4. 劳动者倾向于迁往那些经济发展强劲从而对劳动的需求增加的地方
5. 我们必须注意：区分需求或供给的变动（它表示曲线的移动）与需求量或供给量的变动（它表示沿着曲线的变动）
## 第四章 供给和需求：弹性及其应用
### A. 需求弹性和供给弹性
我们需要知道供给和需求在多大程度上对价格的变动做出反应
#### 需求的价格弹性
1. 需求的价格弹性，有时也称为价格弹性，衡量的是：当一种物品的价格发生变动时，该物需求量相应变动的大小。价格弹性的准确定义是需求量的变动的百分比除以价格变动的百分比
2. 影响价格弹性的因素：物品本身的价格弹性，即对价格变化的敏感程度差别；人们对价格变动做出反应时间的长短（汽油就是一个典型的例子。假设当汽油价格突然上涨，而你正在驾驶着汽车横穿美国大陆，比是否会卖掉汽车放弃旅行？一般情况下你大概不会。因此，在短期内，这种场合的汽油需求可能就非常缺乏弹性。而从长期来看，你可以根据汽油的较高价格来调整自己的消费（行为），可以购买小型节能车、骑自行车、乘火车、搬到距离工作场所较近的地方居住或者与其他人公用汽车，消费模式调整余地较大）
3. 消费模式调整余地较大，通常意味着所消费的物品的长期需求（的价格）弹性要大于短期需求（的价格）弹性
#### 弹性的计算
1. 需求的弹性 = 需求量的变动的百分比 / 价格变动的百分比
2. 富有需求价格弹性
3. 缺乏需求价格弹性
4. 单位需求价格弹性
5. 计算弹性的准确公式应该是
```math
E_D = \frac{\varDelta Q}{(Q_1 + Q_2) / 2} \div \frac{\varDelta P}{(P_1 + P_2) / 2}
```
#### 计算弹性的一条捷径
1. 为了计算非线性需求曲线的价格弹性，首先过该点做切线。然后计算位于该点之下的切线线段长度与位于该点之上的切线线段长度的比值，该比值就是需求曲线上该点的价格弹性
#### 计算弹性的代数方法
1. 代数方法需要涉及：曲线的斜率、特定价格和需求量。其中b为曲线斜率，P0和Q0为某点的价格和需求量
```math
E_D = b(P_0 / Q_0)
```
2. 需求曲线的起点处价格弹性高
3. 斜率并不等于弹性
4. 弹性等于位于直线或曲线在该点切线上的雪球点之下的线段长度与位于该点之上的线段长度二者的比值
#### 弹性和收益
1. 根据定义，总收益等于价格乘以数量（即P * Q）
2. 价格变动对总收益产生的影响
  + 当需求缺乏弹性时，降低价格会减少总收益
  + 当需求富有弹性时，降低价格会增加总收益
  + 当需求具有单位弹性时，价格下跌不会引起总收益的任何变动
3. 价格歧视是指同一种服务对于不同的顾客收取不同的费用的策略行为
#### 供给的价格弹性
1. 供给的价格弹性是供给量变动的百分比除以价格变动的百分比
2. 影响供给弹性的主要因素是行业中增加生产的困难程度
3. 影响供给弹性的另一个重要因素是考察时段的长短
### B. 弹性在主要经济政策问题中的应用
#### 农业经济学
1. 限制产量是政府在单个市场进行干预的一个典型例子
#### 税收对价格和数量的影响
1. 归宿是指对生产者或消费者的真实收入征税的最终经济影响
2. 运用供求原理，我们可以分析谁实际上承担了税负，即税收的归宿在哪里
3. 税负归宿的一般原理：税收归宿指出了税收对生产者和消费者收入的影响。一般来说，税收归宿取决于供给和需求的相对弹性。（1）如果需求相对于供给缺乏弹性，则税收主要是向前转嫁给消费者；（2）如果供给相对于需求缺乏弹性，则税收主要向后转嫁给生产者
#### 总结提要
1. 切勿将弹性与斜率相混淆
2. 干预可能会提高某些群体的利益，但通常会产生市场扭曲和导致无效率
## 第五章 需求和消费者行为
#### 选择和效用理论
1. 效用表示满足。更准确地说，效用是指消费者如何在不同的物品和服务之间进行排序
2. 通常，我们可以将效用理解成一个人从消费一种物品或服务中得到的主观上的享受或有用性。但是你切不可将效用等同于可观测的或可衡量的心里功效或效果。相反，效用是一种科学构思，经济学家用它来解释理性的消费者如何做出决策
#### 边际效用和边际效用递减规律
1. “边际”是经济学的关键术语，通常是指“新增”或“额外”的意思。边际效用是指多消费1单位商品时所带来的新增的或额外的效用
2. 需求理论背后的一个基本理念就是边际效用递减规律。这个规律指出，随着个人消费越来越多的某种物品，他从中得到的新增的或边际的效用量是下降的
#### 效用的数字表说明
1. 在某一消费水平上，边际效用曲线之下的总面积的单位数都必然等于消费单位数相同时总效用曲线的高度的单位数
2. 总效用是从开始处累计起来的所有边际效用之和
3. 现代效用理论渊源于利功主义
#### 需求曲线的推导
#### 等边际法则
1. 等边际法则：我应该这样安排自己的消费，即在每一种物品上所支出的最后1美元，都能给我带来相同的边际效用
2. 等边际法则：最大满足或效用最大化的基本条件是要符合等边际法则，即当花费在任何一种物品上的最后1美元所得到的边际效用正好等于花费在其他任何一种物品上的最后1美元所得到的边际效用的时候，该消费者就会得到最大的满足或效用
3. 消费达到均衡时用于各种物品的每1美元的一般的边际效用被称为收入的边际效用。它衡量的是消费者追加1美元消费时可以得到的额外的效用
4. 公式表达
```math
\frac{MU_{product1}}{P_1} = \frac{MU_{product1}}{P_1} = \frac{MU_{product1}}{P_1} = ... = MU_{Every\quad dollar\quad of\quad income}
```
#### 效用理论的分析发展
1. 基数效用（或可计量的）-->定量
2. 序数效用：只考察消费者对商品组合的偏好排序-->定性
3. 在等边际法则中讨论效用时，我们所假设的是物品可以分割成若干的很小的单位。然而在许多场合，单位物品的不可分割性也很重要，不能被忽略。
4. 当不可分割性起作用时，等边际的均衡数量法则就会让位于非均衡数量法则
#### 替代效应
1. 替代效应：当某一物品的价格上升时，消费者倾向于用其他商品来代替变得较为昂贵的该种物品，从而更便宜地获得满足
#### 收入效应
1. 实际收入是指你的货币收入能够买到的物品的实际数量。当价格上升且后壁收入固定不变时，消费者的实际收入便会下降
2. 收入效应，即物品价格变化通过对消费者实际收入的影响，进而影响消费者对该物品的需求数量
3. 收入弹性，这一术语表示在其他条件（如价格）保持不变的情况下，需求量变动的百分比除以收入变动的百分比
4. 收入弹性 = 需求量变动的百分比 / 收入变动的百分比
5. 高收入弹性：如空中旅行；低收入弹性：如土豆
#### 从个人需求到市场需求
1. 通过把所有消费者的需求量加总，我们可以得到某一物品的整个市场的需求曲线
2. 我们用小写字母（dd和ss）表示个人的需求曲线和供给曲线，用大写字母（DD和SS）表示市场的需求曲线和供给曲线
#### 需求曲线的移动
1. 需求曲线表明一种物品的需求量如何对它的价格变动做出反应。但是，需求也受到其他物品的价格、消费者的收入和特殊因素的影响。需求曲线实在假设这些条件保持不变的前提条件下描绘出来的。但是，如果这些条件发生了变化，则情况又将会如何呢？此时，整个需求曲线会向右方或左方移动
#### 替代品和互补品
1. 如果物品A的价格上升增加了替代品B的需求，那么，物品A和物品B就互为替代品
2. 物品A的价格上升会降低其互补品B的需求，那么，物品A和物品B就互为互补品
3. 介于替代品和互补品之间的是独立品
#### 价格悖论
1. 商品的数量越多，它的最后一单位的相对购买愿望就越小。因此，为什么大量的水只有极低的价格，为什么必不可少的物品，如空气，却成为免费物品，其答案就清楚了。在这两种情况下，正式巨额的数量使其边际效用大大减少，因而降低了这些重要物品的价格
#### 消费者剩余
1. 一种物品的总效用与其市场价值之间的差额成为消费者剩余。之所以会产生剩余，是因为我们“所得到的大于我们所支付的”，这种额外的好处源于边际效用递减规律
## 第五章附录
#### 无差异曲线
1. 无差异曲线：这条无差异曲线上的点，代表消费者所有的无差异的消费组合：每一种组合你都有同等的意愿去得到
#### 替代规律
1. 一种物品越是稀缺，它的相对替代价值就越大，相对于变得充裕的物品，它的边际效用就会上升
2. 无差异曲线的斜率是两种物品的相对边际效用的度量
3. 替代规律指出，当你得到更多的某种物品时，该物品的替代率或无差异曲线的斜率的斜率递减
#### 切点的均衡位置
1. 从几何上看，当预算线的斜率正好等于无差异曲线的斜率时，消费者达到均衡状态
#### 收入和价格的变化
1. 无差异曲线的两个重要应用在于考虑一下两种效应：（a）货币收入变化；（b）两种物品中的一种的价格发生变化
## 第六章 生产和企业组织
我们的目的是了解市场力量是如何决定商品和服务的供给的
### A. 生产理论和边际产量
#### 生产函数
1. 生产函数是指：在既定的工程技术和知识水平下，给定投入之后能够得到的最大产出（投入-产量 函数）
2. 注意，我们的定义所假定的是：企业总是尽力做到有效率的生产。换句话说，在给定投入量的前提下，它们总是企图生产出最大的产量
#### 总产量、平均产量和边际产量
1. 总产量表示生产出来的、用实物单位衡量的产出总量
2. 边际产量实<u>在其他投入保持不变</u>的情况下，由于新增1单位的投入而多生产出来的产量或产出
3. 这种边际产量统计对于理解如何决定工资等其他要素价格非常重要
4. 平均产量，它等于总产量除以总投入的单位数
#### 边际收益递减规律
1. 边际收益递减规律表明在其他投入不变时，随着某一投入量的增加，新增加的产出越来越少。换言之，在其他投入不变时，随着某一投入量的增加，其每1单位投入的边际产量会下降
2. 边际收益递减规律表明了一个很基本的关系。当一种投入如劳动，被更多地追加既定数量的土地、机器和其他投入要素上时，每单位劳动所能发挥作用的对象越来越有限。土地会越来越拥挤、机器会被过度地使用，从而劳动的边际产量会下降
3. 边际收益递减规律是解释为什么亚洲许多国家如此贫穷的关键性因素之一
4. 另外，边际收益递减规律可能并不适用于所有的产量水平。某个最初的劳动投入实际上可能表现出边际产量的递增。这是因为，需要一个最小的劳动量才能走到田间并拿起锄头。尽管有这些例外，但边际收益递减仍是一条在大多场合和都能适用的规律
#### 规模报酬
1. 规模报酬不变、规模报酬递增经济、规模报酬递减
#### 短期和长期
1. 短期为这样一个时期，在该时期里，企业能够通过改变可变要素，如原料和劳动，但不能改变固定要素（如资本）来调整生产
2. 长期定义为一个足够长的时期，以至于包括资本在内的所有要素都可以得到调整
#### 技术变革
1. 我们要区分产品创新和工艺创新，前者指新的或改良的产品被推向市场；后者则指对已有产品采用新的或改良的生产加工技术
2. 工艺创新等同于生产函数的改变
3. 但是，在出现市场不灵的时候，技术腿部也可能会在市场经济中发生
4. 外部收益：消费者不仅能从自己的使用中获益，而且能够从其他人的使用中收益
5. 网络：很多产品本身没有多大用处，只有在和其他产品良和使用时才具有价值。网络的种类很多，取决于它的物理连接，如电信系统、输电网络、计算及集群、管道网和道路网，以及人们使用相互兼容的软件（如Windows操作系统）或同一语言（如英语）时所形成的间接网络
6. 网络市场特性：
  + ”冒尖鼓励型“：赢者通吃
  + 历史因袭：习惯一旦形成很难改变
  + 规模经济
7. 网络市场为了确定早期的领导：
  + 采用渗透定价策略，即向早期用户提供低价
  + 将你的产品和其他已经流行的产品捆绑在一起
  + 质疑你的竞争对手产品的质量或稳定性
  + 斥巨资做广告
#### 生产率
1. 生产率是总产出对加权平均的投入的比率
2. 生产率涉及两个重要的变量是劳动生产率和全要素生产率，前者计算每单位劳动的产量，后者计算每单位总投入（一般包括资本和劳动）的产量
#### 由规模经济和范围经济引起的生产率的增长
1. 生产率增长的原因有：技术进步、规模经济和范围经济
2. 范围经济发生于大量不同产品同时生产比单独生产更有效率的时候
### B. 企业组织
#### 企业的性质
1. 企业由专业化的组织所组成以便管理生产过程
2. 企业的重要功能是：实现批量生产的优势、筹集资金、管理和协调生产过程
3. 企业由专业化机构组成以管理生产过程。生产在企业里进行的原因在于效率通常要求大规模的生产、筹集巨额资金以及对正在进行的活动进行细致的管理与协调
#### 公司
1. 现代公司的核心特征如下
  + 公司的所有权属于那些掌握了公司普通股的所有人
  + 从原则上讲，股东控制他们所拥有的公司。实际上，大公司的股东并不能真正地控制公司，因为他们太分散，根本不能左右拥有实权的公司经理们
  + 公司的经理和董事会拥有指定公司决策的合法权利。他们决定生产什么和如何生产。总之，股东拥有公司，但经理却经营它
2. 公司的股东承担的是有限责任，即他们所承担的公司债务或亏损，不会超过他们最初的出资
3. 公司的收入要交纳两次所得税：首先要作为公司的利润交税，然后则对以红利形式体现的个人收入征收所得税
#### 总结提要
1. 规模报酬反映所有投入的平衡增加对产量的影响
## 第七章 成本分析
### A. 成本的经济分析
#### 固定成本
1. 即使企业的生产量为零也必须支付的费用较固定成本（FC）。有时，固定成本也称为“固定开销”或“沉没成本”
#### 可变成本
1. 可变成本（VC）是随着产出水平的变化而变化的那些成本
2. 我们对这些成本概念做如下总结
  + 总成本（TC）
  + 固定成本（FC）
  + 可变成本（VC）
  ```math
TC = FC + VC
```
#### 边际成本的定义
1. 边际成本（MC）表示由多生产1单位产出而增加的成本
2. IT一般具有这样一种特性，即生产第一个产品非常昂贵，而生产接下来的拷贝却非常便宜。在这个例子中，微软花费了很大的成本来研发IE浏览器，而多销售一个软件的边际成本却几乎等于零。也就是说，微软生产和销售10000001个浏览器的总成本并不比1000000个更多。而只要IE浏览器的边际成本为零，微软就不会因为对外赠送而赔钱
#### 平均成本或单位成本
1. 公式表达：
```math
AC = \frac{TC}{q}
```
#### 平均固定成本和平均可变成本
1. 平均固定成本（AFC）被定义为FC/q
#### 平均成本与边际成本的关系
+ 当MC在AC之下，它将平均成本拉下
+ 当MC在AC纸上，它将AC拉上
+ 当MC等于AC，AC固定不变。在U型AC曲线底部，MC=AC=AC的最小值
1. AC曲线总在最低点被上升中的MC曲线所穿过
#### 收益递减和U形成本曲线
1. 经济学家通常画出U形成本曲线。对于一个U形的成本曲线，成本在最初的阶段是下降的，达到一个最低点，然后再开始上升
2. 通常，在短期，我们会人为资本是固定成本，而人为劳动是可变成本。但事实并不一定如此（例如长期聘用的教授）。当然一般来说，劳动投入还是比资本投入更容易改变
3. 可变要素的边际收益递减意味着短期边际成本的递增。这就说明了为什么边际收益递减导致边际成本在某一点之后上升
4. 我们可以将生产率规律与成本曲线之间的关系总结如下：在短期，当像资本那样的要素固定不变时，可变要素一般表现为开始阶段的边际产量递增和随后出现的边际产量递减。与之相对应的是，成本曲线表现为开始阶段的边际成本递减和在边际收益递减发生之后出现的边际成本递增
#### 最小成本法则
1. 最小成本法则：为了以最小成本生产出一定数量的产出，企业应该购买各种投入，知道花费在每一投入上的每1美元的边际产量都相等时为止。这就有：
```math
\frac{MP_L}{p_L}\quad =\quad \frac{MP_A}{p_A}\quad =\quad ...
```
2. 企业的这一法则完全类似于追求效用最大化的消费者所遵循的原则
3. 替代法则：如果一种要素的价格下降，而所有其他要素的价格保持不变，则企业用现在更便宜的要素替代所有其他要素，直到所有投入的单位美元的边际产量都相等，这便是有利可图的
### B. 经济成本和企业会计
1. 净收益（或利润） = 总收入 - 总支出
### 资产负债表
1. 资产负债表是企业在某一日期的财务状况表。这个表记录的是企业、个人或国家在某一时点上的价值
2. 资产负债表与损益表的一个重要不同在于存量和流量
3. 存量代表变量的水平
4. 流量代表某一单位时间的变化
5. 损益表统计的是一个企业的流入和流出，而资产负债表度量的是在会计年度末的资产和负债的存量
6. 净值 = 资产 - 负债
#### 财务欺诈
1. 健全的会计审计制度，还有政府机构和非政府机构的晶体和审慎，是何等的重要！
### C. 机会成本
1. 做出决定实际上都使我们失去了做其他事的机会。失去的选择被称为机会成本
2. 作出一项决定的机会成本涉及它所有可能的后果，无论它们是否体现为货币的交易
3. 决策具有机会成本，因为在一个稀缺的世界中选择一样东西意味着需要放弃其他一些东西。机会成本指的是错过的最有价值的物品或劳务的价值
4. 会计数字和实际的经济机会成本有时相去甚远
#### 机会成本和市场
1. 在运转良好的市场上，当所有成本都包括进来时，价格等于机会成本
2. 在分析发生于市场之外的交易时，机会成本的概念显得尤其重要
3. 除了外在的货币支出之外，经济成本还包括那些由于各种资源可以利用在其他方面而导致的机会成本
#### 总结提要
1. 平均总成本（AC） 是持续下降的平均固定成本（AFC）与平均可变成本（AVC）之和
2. 机会成本最重要的应用在于非市场产品
## 第七章附录
#### 数值化的生产函数
1. 应该记住生产函数表示在一定时点上给定管理技能和技术知识的条件下所能得到的最大产量
#### 边际产量递减规律
1. 边际收益递减规律表明，当我们增加一种投入而保持其他投入不变时，所增加投入的边际产量至少在一点之后将会逐步下降
2. 边际收益递减规律是由于固定要素相对于可变要素的减少而造成的
## 第八章 完全竞争市场分析
1. 完全竞争市场（中）的行为是一个理想化的市场，所有单个的企业和消费者的影响都太小，以至于无法影响到市场价格。
2. 完全竞争市场是有效率的
### A. 竞争企业的供给行为
1. 在分析完全竞争企业的供给行为时，有两点需要注意：首先是利润最大化；其次完全竞争的世界是由原子式的企业所构成的，且它们都是价格接受者
#### 完全竞争
1. 完全竞争的世界是一个价格接受者的世界。完全竞争企业出售无差异的产品（与行业中其他企业出售的产品相同）
2. 单个竞争者的需求弹性要比整个市场的需求弹性大得多
3. 总结
  + 在完全竞争条件下，有许多规模较小的企业，每个企业都生产相同的产品，并且，每个企业规模都太小，不可能影响市场的价格
  + 完全竞争条件下的生产者面临的是一条完全水平的需求曲线（即dd曲线）
  + 从所追加的每1单位的出售中，企业获得的额外收入刚好等于市场价格
#### 边际成本等于价格时的竞争供给
1. 最大利润的产量发生在边际成本等于价格的产量水平上
2. 在完全竞争条件下企业的供给原则为：当企业将其产量确定在使边际成本等于价格的水平上时，就实现了利润的最大化，即：MC = P
3. 零利润点，它表示在这一价格水平上企业得到的是零利润；在零利润点上，价格刚好等于平均成本，因此收入正好弥补成本
4. 对于一个利润最大化的竞争企业来说，边际成本（MC）曲线向上倾斜的部分就是该企业的供给曲线
5. 一个利润最大化的企业将其产量确定在能使边际成本等于价格的水平上。用图形来说，这就意味着企业的边际成本曲线也是它的供给曲线
#### 总成本与停业条件
1. 即使在企业没有生产任何产品时，它仍须履行契约
2. 只要收入足以弥补可变成本，继续生产就会比较合算
3. 在这个市场价格水平，企业的收入正好等于它的可变成本（或者说，损失正好等于固定成本），这个关键的低市场价格称为停业点
4. 停业原则：企业在收入刚好抵补它的可变成本或者损失正好等于固定成本时，停业点就会出现。当价格低于平均可变成本时，企业就会停业以使其利润最大化（即损失最小化）
5. 只要企业的收入超过它的可变成本，它的供给曲线就与其MC曲线相一致
6. 追求利润最大化的企业即使亏损，也可能在短期内继续经营
### B. 竞争企业的供给行为
#### 加总所有企业的供给曲线可得到市场供给曲线
1. 市场需求曲线DD也是把单个的需求曲线dd按水平方向加总得到的
#### 短期均衡和长期均衡
1. 经济学家发现，需求移动在短期内会比在长期内引起更大的价格调整和更小的数量调整（因为短期均衡，劳动要素供给曲线弹性更小
2. 在可以自由进入和退出某个行业，而且任何数量的企业都可以按同样的、不变的成本曲线进行生产的情况下，长期供给曲线SS等于每个企业的最低平均成本或零利润的价格的一条水平线。如果该行业使用的是某种特殊要素（稀缺的），长期供给曲线将会向上倾斜（由于边际收益递减规律）
#### 一个竞争性企业的长期行为
1. 竞争的加剧是否会最终导致长期的亏损？当企业再也不能弥补它们的可变成本时，就会停止营业。但是，在长期内，所有的成本都是可变的。所以，在长期内企业仅仅在价格等于或高于零利润点（在该点价格等于平均成本）时，才愿意进行生产
2. 在长期内，一个行业的价格趋向于某一关键点，在该点，同行的企业正好能够弥补它们的全部竞争成本
3. 长期的零利润均衡：当一个行业的供给是由具有完全相同成本曲线的竞争企业所提供，而且这些企业又可以自由地进入或退出该行业时，长期均衡的条件就是：对于每一个完全相同的企业来说，价格等于边际成本，也等于最低长期平均成本点、即：P = MC = 最低长期AC = 零利润的价格。这就是长期的令经济利润条件
4. 竞争的力量趋向于把各个企业的和各个行业推向零利润的长期状态
5. 因此，在一个竞争性行业中，长期均衡是一种没有经济利润的均衡
### C. 竞争市场的各种特殊情况
#### 一般原则
1. 几种重要的成本和供给情况：成本不变、成本递增和收益递减、固定的供给与经济租金
2. 当供给数量不随着价格的变化而变化时，对这种生产要素的支付称为租金或纯经济租金
#### 供给的变动
1. 当需求缺乏弹性时，供给的增加使价格P下降的幅度最大
2. 当需求缺乏弹性时，供给的增加使数量Q增加的幅度最小
### D. 竞争市场的效率和公平
本章将对完全竞争市场的效率作一个总体上的评综
#### 效率的概念
1. 在任何可能的生产资源重组都不能在不使其他人的情况变坏的条件下，使得任何一个人的福利变好时，就达到了帕累托效率，或简称为效率。因此，在实现了配置效率的条件下，只有降低某个人的效用才能增加另一个人的满足或效用
#### 竞争性均衡的效率
1. 为了简便起见，我们用闲暇时间的“尤特尔”来衡量福利水平（或用正常工作时间“非尤特尔”）。我们进一步假设每一小时被放弃的闲暇的边际成本不变，所以，所有闲暇与劳动的效用和成本都可以计算出来。
2. 经济剩余 = 消费者剩余 + 生产者剩余
3. 经济剩余是从生产和消费某一种物品中所得到的福利或净效用，它等于消费者剩余与生产者剩余之和
4. 对竞争性均衡的详细分析将证明，它使产业中可能的经济剩余达到最大化
5. 社会从消费最后1单位中得到的边际收益等于社会生产最后1单位的边际成本，从而保证了竞争性均衡的效率
#### 众多消费者和市场的均衡
1. 当个人的边际成本等于社会的边际成本，且二者都等于边际效用时，一个完全竞争的竞技就是有效率的
2. 通过使价格与边际成本相等，竞争就可确保经济达到配置效率
3. 竞争才是效率的保证，这个原则在多种要素和多种产品的情况下同样是成立的
#### 边际成本作为效率标准
1. 运用边际成本来获得生产效率，对于任何一个试图最有效率地利用其资源的社会或组织来说都是适用的
2. 边际成本并非只适用于利润最大化的企业，而适用于各种经济问题，事实上，它适用于所有涉及到稀缺性的问题
3. 在关于效率的分析中，边际成本是一个最基本的概念。对于任意追求某一目标的组织，效率要求它在每一项活动中，为实现目标所投入的边际成本都相等。市场经济中，只有当每家企业的边际成本MC都等于一个共同的市场价格时，该行业才能以最小的成本进行生产
#### 限制
1. 竞争市场具有显著的效率特性
2. 市场的一些缺陷：
  + 当存在污染、其他外部性、不完全竞争和不完全信息时，市场便无效
  + 竞争市场下的收入分配即使是有效率的，也可能不是社会想要的或能接受的
#### 市场不灵
1. 不完全竞争
2. 外部性（不是所有的外部性都有害，有些外部性是有益的）
3. 不完全信息
#### 总结提要
1. 帕累托效率的实现，需要完全竞争、完全信息和不存在外部性的条件。当这三个条件得到满足时，一个重要的效率条件得以成立：价格比率 = 边际成本比率 = 边际效用比率

## 第9章 不完全竞争和垄断
现在，欢迎你来到现实世界，一个不完全竞争的世界
### A. 不完全竞争模式
1. 与完全竞争相比，不完全竞争的价格较高，而产量较低
#### 不完全竞争的定义
1. 如果一个企业能够影响其产品的市场价格，那么，该企业就是一个不完全竞争者
2. 当个别卖者能够影响某一行业的产品价格时，该行业就处于不完全竞争之中。不完全竞争的类型主要有垄断、寡头和垄断竞争
3. 一个不完全竞争者对其价格的确拥有某些但又绝不是全部的拍板权
4. 不完全竞争者是价格的制定者而非价格接受者
#### 形形色色的不完全竞争者
1. 不完全竞争市场结构分为以下三种类型：垄断、寡头、垄断竞争
#### 垄断
1. 垄断：单一的卖者完全控制某一行业（这个单一的卖者称为“垄断者”，“垄断者”这个词来源于希腊语中的“单个”和“卖者”这两个词）。垄断者是他所在行业的唯一生产者。同时，没有任何一个行业能够生产出相近的替代品
2. 在长期内，没有一个垄断者能够确保自己免受竞争的冲击
#### 寡头
1. 寡头的意思是“几个卖者”。从上下文来看，几个可以是两个那么少，也可以是10个或15个那么多。寡头的重要特征是每个企业都可以影响市场价格
#### 垄断竞争
1. 垄断竞争：一个产业中有许多卖者生产有差异化的产品
2. 市场上有许多卖者，但任何一个卖者都没有太大的市场份额
3. 垄断竞争与完全竞争不同的是：由不同企业销售的产品并不相同（即差异化产品）
4. 差异化产品在重要的特征上表现不同。例如，个人电脑在速度、内存、硬盘、调制解调器、大小和重量等各方面具有不同的特征
6. 各种市场结构

| 结构 | 生产者的数目和产品差异化程度 | 一般经济领域 | 企业对价格的控制程度 | 营销的方式 |
| --- | --- | --- | --- | --- |
| 完全竞争 | 许多生产：完全相同的产品 | 金融市场与农产品 | 没有 | 市场交易或拍卖 |
| 垄断竞争 | 许多生产者：产品有许多真正或感官上的差异 | 零售业（比萨饼、啤酒...）个人电脑| 一些 | 广告宣传和产品质量方面的竞争，受到管制的价格
| 寡头 | 几个生产者：产品的差异化很小或没有差异/产品有差异化 | 钢铁、化工.../汽车、文字处理软件... | 一些 | 广告宣传和产品质量方面的竞争，受到管制的价格 |
| 垄断 | 单一生产者：产品没有替代品 | 特许经营（电力、自来水）；微软Windows；专利药品 | 相当大 | 广告 |

7. 不完全竞争并不等于没有竞争；完全竞争并不意味着对抗，而只是表示行业中没有一家企业能影响市场价格
#### 市场不完全竞争的根源
1. 两条主要原因：首先，当大规模生产出现规模效益并降低成本时，一个产业中的竞争者就会越来越少；其次，当出现进入壁垒，即新的企业很难加入某一行业时，也有可能出现不完全竞争
#### 进入壁垒
1. 规模经济是进入壁垒的一种很普遍的类型，除此之外，法律限制、进入的高成本以及广告和产品差异化也会形成进入壁垒
+ 法律限制
  + 关税是托拉斯之母。这是因为政府强加的进口限制能将外国竞争者拒之门外。原因可能在于：某种产品的单个国家市场可能只能支撑两三个企业，而世界市场则足够大，可以支撑很多的生产者
+ 进入的高成本
  + 用户一旦掌握了一种软件以后，就不大愿意去学习另一种
+ 广告与产品差异化
  + 广告可以打开产品的知名度并形成名牌效应
2. 少数几家厂商生产许许多多、形形色色的品牌、型号和产品
3. 产品的多样化可以吸引最大范围的消费者
4. 大量具有差异化的产品使潜在的竞争者很难鼓起勇气加入这个市场
5. 每一种有差异化的产品的需求是如此之小，以至于这些市场不能维持许多企业达到U形成本曲线的最低点进行生产（理解：在完全竞争市场，此时，对于单个企业，市场需求量很大，P = MC = AC（U形成本曲线最低点），每个企业都是价格接受者；而由于差异化产品需求量较小，因而，单个企业可以通过规模效益占领市场，使得P > MC）    
6. 产品差异化和关税一样导致了更高的集中程度和更加不完全的竞争
7. 产品的品牌的产生取决于消费者对该产品的品味和质量的认可和口碑
### B. 垄断行为
1. 垄断行为会导致高价格、低效率和低产出，从而会减少消费者的福利
#### 边际收益和价格
1. 边际收益（MR）的定义是：当销售量增加1单位时，由此而引起的收益的变化。MR既可以为正，也可以为负
2. 负边际收益是指：为了售出更多单位的产量，该企业必须将价格降低到如此低的程度，以至于它的总收益下降
3. 当需求富有弹性时，MR起初是正的。然而，在需求转变为缺乏弹性之后，尽管价格仍是正数，MR却变为负数
4. 当dd曲线变为缺乏弹性时，MR变为负数
#### 弹性和边际收益
1. 需要牢记的要点如下：
  + 边际收益（MR）是销售增加以单位时所引起的收益的变动
  + 价格 = 平均收益（P = AR）
  + 当需求曲线向下倾斜时，P > MR = P - 此前销售的所有的单位的收益的减少量
  + 当需求有弹性时，边际收益为正；当需求为单位弹性时，边际收益为零；当需求缺乏弹性时，边际收益为负
  + 对完全竞争者而言，P = MR = AR
2. 需求弹性、产量、价格、收益及边际收益之间的关系

| 需求 | 产量（q）和价格（P）之间的关系 | 产量（q）对总收益（TR）的影响 | 边际收益（MR）|
| --- | --- | --- | --- | 
| 富有弹性（Ep > 1） | q变动百分比 > P变动百分比 | q越高，TR越高 | MR > 0 |
| 单位弹性（Ep = 1） | q变动百分比 = P变动百分比 | q增大，TR不变 | MR = 0 |
| 缺乏弹性（Ep < 1） | q变动百分比 < P变动百分比 | q越高，TR越低 | MR < 0 |
#### 利润最大化的条件
1. 总利润 = 总收益 - 总成本，符号表示： TP = TR - TC = (P * q) - TC
2. 当产量达到该企业的边际收益等于它的边际成本的水平时，利润达到最大，即 MR = MC
#### 垄断均衡的图示
1. 当产量确定在MC = MR的水平，垄断者就可以使其利润最大化。由于该垄断者拥有一条向下倾斜的需求曲线，这就意味着 P > MR。对于一个追求利润最大化的垄断者来说，由于价格高于边际成本，因此，垄断者会将产量水平降低到低于在完全竞争产业中所决定的水平
#### 完全竞争作为不完全竞争的一个极端情况
1. 完全竞争者的 P = MC，是 MR = MC 利润最大化条件的一个特例
2. 利润最大化原则：MR = MC
#### 边际原则：让过去成为过去
1. 你应该注意决策的边际成本和边际收益，并忽略过去的或沉没的成本
#### 损失规避和边际原则
1. ”损失规避“意思是说即使继续持有某项资产要付出昂贵的代价，人们也拒绝承担现时的损失。这种行为已经在房市中得到证实，在房地产市场上，人们在售房遭遇损失时总是开出更高的要加，并且会等上更长的时间
2. ”托拉斯“，即股东将股票转交给托管人，由他们负责经营，实现利润最大化
## 第10章 寡头和垄断竞争
### A. 不完全竞争者的行为
#### 市场力量的衡量
1. 市场力量指的是单一厂商或少数企业控制某一行业的价格制定和生产决策的程度
2. 衡量市场力量最常用的一个指标是一个产业的集中率
3. 四企业集中率的定义是：某产业四家最大的厂商的总产量（或发货量）在整个产业中所占的百分比
4. 八企业集中率就是前八大厂商在某个产业总产量中所占的百分比
5. 要想充分了解那些占统治地位的企业在行业中的角色，赫芬达尔-赫希曼指数（Herfindahl-Hirschman Index,HHI）是一种更好的方法。该指数是通过将一个市场中所有参与者所占的市场份额的比率的平方进行加总而获得的
6. 慎用集中率指标的情形：
  + 国际竞争
  + 关系密切的行业之间的竞争：集中率指标往往忽略了来自其他相关产业的竞争（例如：有线电话和微信电话）
#### 不完全竞争的实质
1. 不完全竞争市场存在着三个主要的因素：经济规模、进入壁垒和策略互动
2. 策略互动是寡头市场的一个新特征，属博弈论研究领域。当每个企业的经营战略取决于它的竞争对手的行为时，就会出现策略互动
3. 集中行业的利润率比非集中行业并高不了多少
#### 不完全竞争理论
1. 我们将考虑不完全竞争的三种重要情况：勾结寡头、垄断竞争和少数寡头
#### 有勾结的寡头
1. 策略互动描述的就是每个企业的经营策略如何依赖于竞争对手的商业行为
2. 勾结：两个或更多的企业共同确定它们的价格或产量，在它们之间瓜分市场，或者共同制定其他生产决策
3. 卡特尔是生产相似产品的独立企业联合起来以提高价格华为限制产量的一种组织
4. 公司相互勾结起来共同确定价格或瓜分市场的行为属于非法的
5. 多家企业厌倦毁灭性的价格大战，于是暗中勾结，将价格定在同一水平，而不是相互压价。在这种情况下，企业可能需要寻求一种勾结寡头均衡，以得到使其共同利润最高的定价。这样这四家企业实际上成了一个垄断者
6. 事实上，经验表明，成功运作卡特尔是一件很难的事，不管是公开的还是秘密的
7. 虽然许多寡头对获取高利润感到满意，但是，在现实生活中，存在许多阻碍它们有效勾结的因素。首先，勾结是非法的。其次，企业可能通过对所选择的顾客降低价格以增加其市场份额来”欺骗“协议中的其他成员。在价格保密、产品有差别、企业的数目不只几个或技术变化迅速的市场上，秘密降价的可能性更大。第三，随着国际贸易的不断深入，许多公司不仅要应付国内竞争，还要迎接外国企业的激烈挑战
#### 垄断竞争
1. 垄断竞争在三个方面类似于完全竞争：有许多买者和卖者，进入和退出某一产业是自由的，各企业都有把其他企业的价格视为既定。二者之间的差别在于：在完全竞争下，产品时完全相同的；而在垄断竞争的条件下，产品时有差别的
2. 产品存在差别意味着每个销售者相对于完全竞争市场（销售者是价格接受者）来说在某种程度上都有提高或降低价格的自由。产品差异化使得每个卖者所面临的需求曲线向下倾斜
3. 随着拥有新差异化产品的企业的进入，这种不完全竞争行业的长期利润率为零（由于新的竞争者的进入，导致需求曲线左移导致）
4. 在垄断竞争的长期均衡点，价格高于边际成本，但经济利润已经下降到零
5. 所生产的大量产品满足了不同消费者的品味和需要。减少垄断竞争者的数目可能会降低消费者的福利，因为人们再也不能得到如此多样化的产品了。人们宁愿为自由选择支付较高的代价
#### 价格歧视
1. 价格歧视指的是同样的产品以不同的价格卖给不同的顾客（例如对新老用户价格策略不同，降低老客户福利（老客户对价格的弹性小，需要更多的转移成本），转移到新客户促销上、对非vip用户阉割产品功能）
### B. 博弈论
1. 博弈论所分析的是，目标相互冲突的两个或更多的决策者的行为及其互动的情况
2. 在不完全竞争的市场结构中，博弈论者若干重要的见解如下：
  + 随着不合作寡头数量的增加，一个产业的价格和产量将接近倾向于完全竞争市场的产出情况
  + 如果企业成功勾结，那么，市场价格和产量将接近于垄断所产生的价格和数量
  + 过去的经历表明，随着企业数量的增加，勾结协议更加难以付诸实施，发生欺骗和不合作行为的次数也会增加
  + 在多种情况下，不存在寡头市场的稳定均衡。由于企业威胁、欺骗、发动价格战、处罚较弱的对手、暗示它们的意向或简单地退出市场等原因，策略互动就可能导致不稳定的后果
3. 博弈论所分析的是两个或两个以上的比赛者或参与者选择能够共同影响每一个参加者的行动或策略的方式
#### 定价策略
1. 一旦你开始考虑他人会如何的对自己的决策做出反应的时候，你便进入了博弈论的领域
#### 基本概念
1. 体现两个企业或两个人之间相互作用的有用工具是双方的回报矩阵
#### 策略选择
1. 博弈论的指导思想是：假设你的对手在研究你的策略，并采取追求自身最大利益的行动，这时你应该如何选择最有效的策略
2. 占优策略，即无论其他博弈者采用何种战略，该博弈者的策略总是最好的
3. 在两个（或全部）博弈者都采用占优策略时，我们称其结果是一种占优均衡
4. 纳什均衡：每个人（博弈的双方或多方）都选择对自己受益最大的策略时，所达到的均衡
5. 由于每一企业都想”欺骗“对手并通过降价获取更多利润，因而确保了在无合谋的情况下，正常价格的纳什均衡将成为一个普遍的事实
6. 博弈论的一条基本原则：把自己的战略建立在嘉定对手会按其最佳利益行动的基础上
7. 纳什均衡是一个在其他博弈者的策略给定时，没有一方还能改善自己的获利情况。纳什均衡有时也被称作非合作性均衡。这是因为，每一方选择策略时都没有合谋，他们只是选择对自身最有力的策略，而不考虑社会福利或任何其他群体的利益
### C. 公共政策如何应对市场力量
#### 提高价格和产出不足的代价
1. 垄断者减少产量和提高价格，因此它们的产量低于应有的像在完全竞争行业中的那种水平
2. 通过保持产品的稀缺性，垄断公司将其价格提高到边际成本之上（P > MC）
#### 不完全竞争的静态成本
1. 净损失，这一术语用来衡量缺乏效率所造成的经济危害
#### 应对不完全竞争的公共政策
1. 国家如何降低垄断带来的破坏性影响：
  + 经济管制
  + 反托拉斯政策
  + 鼓励竞争：对一些幼稚企业发展的鼓励；从国际市场向国内市场引入竞争
#### 反托拉斯法与经济学
1. 反垄断法通过两种不同的方式打击有碍竞争的不当行为：首先，反垄断法禁止某些类型的商业行为，比如抑制竞争的价格固定行为。其次，反垄断法限制某些类型的市场结构，比如垄断
#### 结构：越大越坏吗
1. 尽管通过公平手段获取的垄断地位是合法的，但采取行动遏制竞争确实非法的
#### 反托拉斯法与效率
1. 对反托拉斯政策热态度的发生转变的原因何在？首先，经济学家发现，集中度高的行业有时会有出色的表现。也就是说，尽管静态地看，集中度高的行业效率低；然而动态地看，这些行业却具有较高的效率，并且其动态效率要远远超过静态的低效率。考虑英特尔、微软和波音公司，它们拥有可观的市场份额，但是它们仍保有高度的创新性，并在商业上取得了巨大的成功
2. 许多经济学家都认为，就强化市场纪律而言，来自外国竞争的威胁，要远胜过反托拉斯法这一工具的威力
## 第11章 不确定性经济学
1. 不认真研究关于不确定性下的决策和信息经济学的奇妙问题，就不可能完整地把握经济生活的现实
### A. 风险和不确定性经济学
#### 投机：资产或商品的跨时空调配
1. 投机包括对有价值物品或商品的买和卖，是从市场价格的波动中谋取利益的一种哦活动。投机者希望能够低价买入，高价卖出
2. 许多人认为，投机是一种不太好的行为，尤其是在利用财务造假或刺探内部消息进行投机的时候，但是投机者有助于熨平商品在不同地区和不同时期的价格，最终能消除各种价格差异。投机者在商品充裕、价格低廉的时候买进，而在商品紧俏、价格高昂的时期卖出。投机者就是经由这种方式去消除价格在时间和空间上的差异，从而在事实上提高了整个市场的效率
#### 套利和价格地理格局
1. 最简单的投机活动是通过买卖同一商品降低或消除地区差异。这种活动被称为套利，即在一个市场上买入一种商品或资产，然后马上在另一个市场上卖出，通过一买一卖的价格差获取利润
2. 由于套利的结果，市场之间的价格差通常不会超过商品在市场之间的转移成本
#### 通过套期保值分摊风险
1. 投机市场的一个重要功能是让人们通过套期保值分摊风险
2. 套期保值是通过对所拥有的一种商品做对冲交易来降低风险。让我们来看看它是如何操作的。假设一位女士拥有一座谷物仓库。她在秋季在堪萨斯买进200万蒲式耳玉米，储存6个月，在第二年春天卖出，每1蒲式耳的溢价为10美分，正好抵偿所有的成本。但谷物的价格可能波动，一旦谷物的价格严格下跌，会赔尽她所有的利润。于是这位仓库拥有者的套期保值就是：在从农民手中买进玉米的同时立即卖出，而不是等到6个月以后交货的时候。当她在9月份买进200万蒲式耳玉米时，立即进行远期交易卖出，协议的未来交易价格使得她恰好能得到每蒲式耳10美分的存储费用
3. 套期保值通过做对冲交易来回避价格波动的风险
#### 投机的经济影响
1. 我们发现，完美的投机行为实现了商品从其数量颇丰（价格低）时期向数量稀缺（价格高）时期的重新配置
2. 我们的讨论表明，理想的投机市场可以提高经济效率
3. 尽管金融投机、套期保值和套利所涉及的资产价值（金融资产例如：股票、债券等）要远大于商品投机活动的规模，但前者的基本原理和后者是完全一样的
4. 理想的投机对于减少消费的不利波动有重要的作用。在一个人人都厌恶风险的现实世界中，投机可能提高总体效用和分配效益
#### 风险和不确定性
1. 若一个人为损失一定量的收入而产生的痛苦感大于他为得到同等数量的收入而产生的满足感，则他就是一个风险规避者
2. 在同样的平均值条件下，人们宁愿要不确定性小的结果。由于这个原因，降低消费不确定性的活动能够导致经济福利的改善
3. 赌博与投机是完全不同的。理想的投机行为可以提高经济福利，而赌博却会带来严重的经济问题
4. 从长远看几乎所有的当事人都要输钱，因为赌场总要从赌注中拿走一份
### B. 保险经济学
1. 市场机制通过风险分摊来应付各种风险。这一过程就是将对一个人来说可能是很大的风险分摊给许多人，从而使每个人所承担的风险很小。风险分摊的主要形式是一种方向相反的赌博形式--保险
2. 保险公司通过集中各种不同的风险来分摊风险，也就是说，它为数以百万计的住宅、生命或汽车提供保险
3. 保险公司的优势在于，对个人来说是难以预料的事件，对整个人群来说则是具有很强的可预见性
4. 保险将最大的风险切分成小块，并零星地出售它们以换取小额的风险酬金。虽然保险可谓另一种形式的赌博，实际上却能正好发挥出一种（与赌博）相反的作用。尽管自然界会造成风险，但保险却能够帮助我们减轻和分散这些风险
#### 资本市场与风险分摊
1. 有形资本的资金所有权可以通过企业财务所有权这个媒介，将风险在很多所有者之间进行分摊
2. 通过将风险性资产的所有权分散到众多的所有者身上，资本市场便可进行分散风险，并且能够鼓励人们进行规模大得多的投资，而这样的投资及其风险，对于单个所有者来说也是可以接受的
#### 信息经济中的市场不灵
1. 市场不灵的两个重要因素是：逆向选择和道德风险
#### 道德风险和逆向选择
1. 如果能满足理想条件:存在多种投保情况，所有投保的风险或多或少地具有独立性，随机事件的概率也可以准确地做出估计，且个人利益未能腐蚀掉人们的基本道德，那么，私人保险市场就能够有效地运转
2. 有时候，逆向选择会导致私人保险机制的缺失。当风险最大者成为最有可能购买保险的人群的时候，就会发生逆向选择问题
3. 当买卖双方在重要的事实上所了解的信息不同：例如个人的健康状况或者一个正在出售商品的质量，就会发生信息不对称问题
4. 当买卖双方存在信息不对称时，类似的市场不灵尤为严重
#### 社会保险
1. 当市场不灵严重以致私人保险不能提供足够的保险项目时，很可能需要政府提供社会保险。它包括覆盖大部分或全部地区的各种强制性项目，由税收和社会保险收费来筹资
2. 扩大保险的覆盖面以避免逆向选择
3. 让我们花些时间来看看失业保险的例子。这是一个私人保险市场无效的例子，因为私人保险所必需的诸多前提条件均不能满足：道德风险（如果保险金很高的话，人们会选择主动失业）、逆向选择（经常失业的人最愿意参加）、失业风险不存在独立性（它们将会伴随经济衰退出现），经济周期又无法预测，所以当失业风险不能被精确地度量。同时，一些政权当局都认为，当人们失业时，势必需要一张社会安全网。因此，政府通常都介入和提供失业保险
4. 当私人保险市场不能有效发挥作用，而且社会相信个人应该有社会安全网，以应对严重的风险（如失业、疾病和低收入）时，政府就会出面提供社会保险
### C. 医疗保障：不会消失的问题
#### 作为社会保险项目的医疗保险
1. 为什么医疗保险是一项社会保险？健康经济专辑给出了这样三个原因：
  + 许多医疗保障系统，例如预防传染病和基础科学的发展，都是市场无法提供的公共品
  + 市场不灵的第二种情况是私人保险市场不灵。这种不灵的最大原因之一是病人、医生和保险公司之间存在信息不对称
  + 政府政策的第三个考虑因素是公平，即应当为所有人都提供最低标准的医疗服务
#### 分配医疗
1. 而在许多医疗领域，我们却不能允许价格机制将医疗服务分配给出价最高的人
2. 医疗服务补助会导致短缺，对这种需求必须通过其他方式限制（例如排队）。这一现象被称为”非价格配给“
### D. 创新与信息
1. 信息与一般商品（如比萨饼和鞋）的差异很大，因为它生产起来花费巨大，但复制起来却成本很小。正是由于信息的这种不同寻常的特性，导致很易受市场不灵的影响，因此，我们需要制定不同的政策来管理信息，如关于”知识产权“的各种法规
#### 信息经济学
1. 公司不能占有其发明者的全部货币价值的情况称为不可划拨性。发明不能被完全划拨是因为，其他公司可能会模仿或非法翻印一项创造品
2. 发明的社会效益（也就是发明对于所有消费者和生产者的价值）
#### 知识产权
1. 知识产权的目的是保护所有者的资源，以免他人在不对所有者或最初发明者进行补偿的情况下进行复制和使用
2. 知识产权最早的形式之一就是专利，即由政府授权在有限的时间内，排他地（有效地，有限垄断地）使用一项”新颖、并非很起眼但很有用“的发明成果
#### 互联网的两难境地
1. 必须注意的一点是，完全竞争在这里是无法存在的，因为若价格等于边际成本，即价格为零，则企业的收入为零，从而无法生存
2. 当今世界日益致力于开发新知识（其中很多是无形的，例如音乐、电影、新的药物和软件），政府必须为知识产权的流通找到一个中介平台。若知识产权保护过强，则会导致提高价格和垄断损失；而若保护过度，则又会阻碍发明和创新
## 第12章 市场如何决定收入
### A. 收入与财富
#### 收入
1. 收入指的是一定时期内（通常为一年）的工资、利息、股息和其他有价物品的流入
2. 所有收入的总和就是国民收入
#### 政府的作用
1. 各级政府以转移支付的形式提供收入
2. 转移支付是政府向个人提供的一种不需要用商品或服务加以回报的支付。转移支付中最大的一项就是对老年人的社会保险，此外还包括失业保险、农业补贴和福利开支
3. 个人收入等于市场收入加上转移支付。市场收入大部分来自工资和薪金：少数富人的市场收入来自于财产收益。政府的转移支付主要用于支付老年人的社会保障
#### 财富
1. 财富是人们在某一时间点所拥有的资产的货币净值。注意，财富是一个存量（如同湖中的水量），而收入是单位时间内的流量（如同河中的水流）。
2. 家庭的财富包括有形财产（房屋、汽车、其他耐用品及土地），还有金融资产（如现金、储蓄、债券和股票）。所有有价值的东西都叫做资产，而所有欠他人的东西都称为负债。总资产和总负债的差额称为财富或净财富
### B. 边际生产率决定投入的价格
1. 收入分配理论或分配理论研究的是经济中收入的决定问题
2. 财富分布的中位数值要远远小于其平均数值，反映了人们拥有的财富的极大不平等
3. 收入分配理论是一般价格理论的特例
4. 供给和需求知识理解竞争性市场经济中收入分配问题的第一步，我们将会看到，收入的关键在于不同生产要素的边际产出
#### 要素需求的性质
1. 要素需求与消费品需求的主要差别有二：（1）要素需求是派生需求；（2）要素需求是相互依赖的需求
#### 要素需求是派生需求
1. 普通消费者的需求和企业对投入品的需求之间有着本质的差别：消费者需要计算机游戏和比萨饼等最终产品，是因为这些消费品都能直接地提供快乐或效用。而像企业购买办公场地这类投入，却并不是因为它们能直接地提供满足；企业购买投入品是因为它能够通过使用这些要素而得到最终产品和相应收入
2. 是消费者的需求最终决定了企业对办公场地的需求
3. 企业的投入需求由消费者对其最终产品的需求间接地派生而来
#### 要素需求相互依赖
1. 某种要素的生产率，如劳动生产率，取决于能够与之相匹配的要素的数量
2. 这意味着在一般情况下，我们不应该说某一种投入独自创造了多少产出。说哪种要素更重要就好比说生孩子是父亲重要还是母亲重要一样
3. 正是由于土地、劳动和资本品在生产中是相互依赖的，才使得收入的分配成为一个非常复杂的问题
4. 如何将每种投入单独的贡献从整体分离出来呢？为了群求答案，我们需要考察要素的边际生产率（它影响需求）和要素的供给之间的相互作用关系。者二者决定要素的市场价格和交易数量
5. 一种投入品的边际产量是指：在其他投入品不变时，增加一单位某种投入品后所能带来的产品或产出的增加量
#### 分配理论和边际收益产品
1. 对各种要素产品的需求来自每种要素通过其边际产品所产生的收入
#### 边际收益产品
1. 边际收益产品（MRP）用来衡量每一单新增投入品所能增加的货币收入。经济学家将增加以单位投入所能增加的产出的货币价值叫做”边际收益产品“
2. MRP = MR * MP
3. 投入品A的边际收益产品是追加一单位投入A所能产生的货币收益
4. 在完全竞争条件下，每一个工人对于企业的价值都等于最后一个工人编辑产品的价值；每一英亩土地的价值等于土地的边际产品乘以产出的价格；对于其他各种要素可以依此类推
5. 在不完全竞争条件下，每个企业所面对的需求曲线是向下倾斜的。此时，从出售的每一单位新增产出上获得的边际收益低于价格，因为企业为卖出1单位新增产出必须能降低此前所有产品的价格。对于企业来说每单位编辑产品的价值 MR < P
#### 追求利润最大化的厂商的要素需求
1. 要达到利润最大化，企业应当增加投入，直到投入的边际效益产品与投入的边际成本或价格相等
2. 最低成本法则：当每一美元投入的边际产品都相等时，产品就达到了最小。这对于完全竞争和不完全竞争条件下的产品市场上的厂商都同样适用
```math
\frac{MP_1}{MC_1} = \frac{MP_2}{MC_2} = ...
```
3. 由最小成本法则进而得到利润最大化意味着：
```math
\frac{MP_1}{MC_1} = \frac{MP_2}{MC_2} = ... = \frac{1}{MR}
```
4. 劳动的边际产品 * 产出价格（MR）= 劳动的价格 = 工资
5. 土地的编辑产品 * 产出价格 = 土地的价格 = 地租
#### 边际收益产品和要素需求
1. 不同厂商需求曲线的加总是决定劳动和其他要素需求的最后一步
2. 替代法则和对要素的派生需求对于完全竞争和不完全竞争的产品市场均适用
#### 生产要素的供给
1. 劳动供给是由许多经济因素和非经济因素所共同决定的，重要的决定因素有劳动的价格（即工资率）和人口特征，如年龄、性别、教育和家庭结构
#### 供给和需求决定要素价格
1. 在给定的要素价格下，追求利润最大化的厂商将按各种投入的边际收益产品选择投入组合
2. 完全竞争市场中投入的均衡价格，即是供给量和需求量相等时的那个价格水平
3. 外科医生和快餐工人收入悬殊的原因在于劳动质量的差别，而不是工作时间的问题
4. 是财富的多少而不是回报率高低导致财富财产收入的差距如此之大
#### 国民收入的分配
1. 在完全竞争条件下。如果市场上工人工资已经超过工人的边际成本，土地所有者就不会再雇佣工人。因此，竞争会确保所有的工人获得的工资率与最后一个被雇佣的工人的边际产品相等
#### 多种投入的边际生产理论
1. 在竞争市场中，投入的需求是由要素的边际产品决定的
2. 收入分配的总体理论与多重要素生产的商品的竞争性定价是相容的。这个简单有力的说明：在一个竞争市场经济中，收入分配如何与生产率相关联的
#### 收入分配中是否也存在看不见的手
1. 在最有效地提供种类不断丰富的产品和服务方面，市场能够发挥强大的作用，然而并不存在一只看不见的手来保证自由经济能够公平地分配收入和财产
## 第13章 劳动市场
### 工资决定的基本理论
#### 一般工资水平
1. 平均实际工资，它代表1小时工作所能得到的实际购买力，或是除以生活费费用的货币工资
#### 边际生产率差异
1. 对于一种生产要素的需求反映了那种投入的边际生产率
#### 国际比较
1. 资本、教育水平、技术水平进和国内竞争竞争等其他一些因素对供应曲线有巨大的影响
2. 不同国家之间的工资差异是由劳动的供给和需求的作用而形成的
3. 墨西哥的工资比美国要低。从根本上说，是因为在墨西哥，劳动的边际生产率低
#### 供给的决定因素
1. 劳动的供给指的是人们愿意在有收益的活动中工作的小时数。决定劳动供给的的三个主要因素是：每个劳工的工作时间、劳动力参与程度和移民状况
2. 导致美国高工资的主要因素美国劳动力受过更好的教育，工作技能更加熟练，供每个劳工使用的资本更多，技术更加现代化
3. 替代效应：因为每小时工作的工资比以前更多，这样每小时的闲暇就变得比以前更昂贵，于是你会受到一种激励，想用额外工作去代因为每小时工作的工资比以前更多，这样每小时的闲暇就变得比以前更昂贵，于是你会受到一种激励，想用额外工作去代你去替闲暇
4. 收入效应：有了更多的收入，你就会想购买更多的商品和服务，此外你还想有更多的闲暇时间。你能够享有更长的假期或更早退休，而在以前你却不能做这些事
5. 哪一种效应更大，是替代效应还是收入效应？没有唯一正确的答案，这取决于个人
#### 实证发现
1. 对于作为一个整体的人口而言，劳动供给对实际工资变化的反应似乎非常小
#### 工资差异
1. 为了解释产业之间和个人普遍存在的工资差异，我们必须考虑到工作之间的差别、人与人之间的差别以及劳工市场上的不完全竞争
#### 工种之间的差别：补偿性工资差异
1. 各种工种的吸引力不同，因此必须提高工资以诱导人们进入这些吸引力较小的工种
2. 为补偿相对吸引力或货币因素的差别而产生的各种工种之间的工资差异，称为补偿性差异
#### 劳工之间的差异：劳动质量
1. 工资差异的一个关键因素，在于人们的劳动质量存在着巨大的差别
2. 尽管劳动质量的很多是由非经济因素决定，但积累人力资本的决策还是可以用经济标准来加以衡量。
3. 人力资本这一概念指的是，人们在其接受教育和培训过程中积累起来的有用的和有价值的技术和知识
4. 在今天的服务经济中，公司门处理的更多是信息，而不是原始材料。在信息时代，大学教育中所获得的知识是获取高收入工作的先决条件
5. ”机会只偏爱有准备的头脑。“在一个技术日新月异的世界中，教育能教人理解新的环境并从中获利
#### 人们的差异：独特个人的”租金“
1. 这些天才人士都拥有一种在当今经济中很有价值的技能。在其拥有特殊才能的领域之外，他们只能整到其收入的一部分
2. 经济学家将工资高于他们在次优职业中取得的收入的部分称为纯经济租金；这些收入在逻辑中上等于固定数量的土地获得的租金
#### 分割的市场和非竞争性群体
1. 我们仍会发现在工资之间还存在巨大差距，其主要原因是劳动市场被分割，形成了一些非竞争性群体
2. 当人们专门从事某一特定职业时，他们就成为一个特定劳工亚市场的一部分
3. 竞争性工资的决定因素

| 劳动条件 | 工资结果 |
| --- | --- |
| 人员相同--工作同质| 没有工资差异 |
| 人员相同--工作的吸引力有别 | 补偿性工资差异 |
| 人员不相同，但每种劳动的供给不变（非竞争性群体） | 反映分割的市场上供求的工资差异 |
| 人员不相同，但各竞争群体之间有若干流动性 | 一般供求决定的工资差异的一般均衡范式 |
### B. 劳工市场的问题与政策
1. 不完全竞争的原因之一就是工会，它代表了很大一部分劳工，尽管该比重正在下降。劳工市场的一大问题就是劳工歧视，虽然现在已经比几十年前有了改善，但其仍然是一个值得关注的重要问题。作用域劳工市场的另一个问题就是政府政策，通过设定最低工资、或鼓励或限制工会、宣布歧视非法等举措，政府可以对劳工市场施加有力的影响
#### 美国工会的历史和时间
1. 工会成员的工资和福利是由集体协议决定的。集体协议是劳资双方的代表为达成双方都能接受的雇用条件而进行交涉和商谈的过程
#### 工会如何提高工资
1. 工会通过获得对某一企业或产业劳动供给的合法垄断，从而拥有了市场力量
2. 当工会将工资标准定得很高时，它并没有直接减少劳动供给（它不同于总需求不足所导致的失业）
#### 工会化提高了工资吗
1. 考虑了工人本身差距的情况下，经济学家得出如下结论：工会工人的工资比非工会工人的工资平均高出10%-15&
2. 在剔除了经济周期对劳动所占份额的影响后，我们看到工会华对于美国工资占国民收入的份额没有什么显著的影响
#### 工会与古典失业
1. 工会化产业的工资相对于非工会化产业的工资会上升：同时工会化产业的就业会下降，非工会化产业的就业会上升
2. 古典失业是由于实际工资高于竞争水平而造成的
3. 经济学家经常将古典失业与经济周期中出现的失业（常称为”凯恩斯失业“）进行对比。凯恩斯失业来自于总需求不足
4. 陷入实际工资过高状态的时候，一个国家或地区就会出现高失业率。这种失业对旨在增加总收入的传统宏观经济政策不会做出反应，因而需要通过降低实际工资来解决问题
#### 歧视的定义
1. 当收入差距的产生仅仅是由于不相关的个人特征，如种族、性别、性倾向或宗教等原因时，我们称之为歧视。通常，歧视可以有两种含义：（a）对不同的人一局个人特征而基于不同的待遇；（b）对某一特定群体设置不利的规则（如考试）
#### 统计性歧视
1. 统计性歧视，即个体的情况都被按其所属群体的平均情况而非个人特征加以处理
2. 传统上妇女被排除在工程师等数学能力要求高的职业之外。结果，妇女在报考学校和择业时容易倾向于人文社会科学，这个结果反过来又会强化人们认为妇女对工程不感兴趣的成见
3. 统计性歧视不仅能将个人的群体特征类型化，而且还能减弱个人对教育和培训进行投资的激励，从而反过来又强化关于原有群体特征的成见
#### 减少劳工市场歧视
1. 平权措施。它要求雇主证明：他正采取特别措施以安置和雇用那些少数群体
2. 然而，确保承诺行动近年来已经被广泛地批评为一种”反向歧视“，一些州已经禁止它在就业和教育中加以使用
## 第14章 土地、自然资源和环境
1. 本章将继续分析生产要素、讨论市场机制在土地、自然资源和环境等方面如何发挥作用
### A. 自然资源经济学
1. 在本章我们将简要地介绍对稀缺的自然资源进行定价和对环境进行管理的若干观点
#### 资源种类
1. 什么是重要的自然资源？它们包括土地、水和大气
2. 在分析自然资源时，经济学家常从两个角度区分它们。其中最重要的区分是资源是可分拨的还是不可分拨的。
3. 当厂商或消费者能够获得商品的全部经济价值时，商品被称为可分拨资源。可分的自然拨资源包括土地、矿产资源，以及森林
4. 不可分拨资源是其成本或收益不能完全归属其拥有者的资源，换句话说，不可分拨的资源是一种具有外部性的资源
5. 当市场参与者不能完全获得使用自然资源的成本或收益时，外部性就会发生，市场就会提供错误的信号和扭曲的价格。一般来说，对于外部不经济的产品，市场会生产过度；而对于外部经济的产品，市场又会产生不足
6. 运用资源的方式，取决于资源本身是可再生的还是不可再生的
7. 不可再生资源是指哪些供给量基本固定的资源
8. 可再生资源，它们的效用能够被有规律地补充，而且只要管理得当，它们就能产生无穷无尽的效用
#### 固定的土地和租金
1. 土地的一个不寻常的特征是：数量固定，对价格完全缺乏弹性（这句话是由局限的。因为有时可以排掉湿地中的水使之成为土地，也可以对浅海湾进行填土而”生产“土地。从1630年到1900年，波士顿的土地面积增加了两倍。而且，土地可被用于不同的目的，在世界上许多农业用地已经转变为城市用地了）
#### 作为固定要素收入的租金
1. 固定要素的价格称为资金，或纯经济租金。经济学家所说的”租金“，不仅是就土地要素而言，而且还用它来讨论所有供给固定的要素
2. 一定要记住”租金“一词在经济学中特殊而具体的含义，即租金是对使用供给固定的要素的支付。日常生活中的”租金“通常包括其他的意思，如对租用公寓或建筑所支付的费用
3. 租金（或纯经济租金）是对使用供给固定的生产要素所支付的报酬
4. 对要素的需求是由对要素所生产的产品的需求所派生出来的
5. 因为土地的攻击没有弹性，且土地总是为了它能生产的产品而发挥作用，所以土地的价值完全是由其能够生产的产品的价值派生而来，反之则不成立
#### 对土地征税
1. 对纯经济租金征税不会导致扭曲或无效率
2. 拉姆塞税收：如果一种商品的供给或需求高度缺乏弹性，对该部门征收税将不会对于生产和消费产生什么影响，所导致的扭曲将会非常小
### B. 环境经济学
1. 在这一节中，我们将揭示环境的外部性特征，说明为什么它们会造成经济的无效率，并分析可能的解决方式
#### 外部性
1. 外部性这个概念，其含义是一种向他人施加不被感知的成本或收益的行为，或者说是一种其影响无法完全体现在它的市场价格上的行为
#### 外部性引起的市场无效率
1. 亚伯拉罕林肯曾说过，政府应当”为人们做那些他们想做，但仅凭个人力量又根本无法做到或做好的事情“
#### 对无效率的分析
1. 符合社会效率标准的污染：达到社会效率标准需要控污的社会边际收益等于控污的社会边际收入
2. 在许多场合，我们发现当一种行为的边际成本等于边际收益的时候，该行为的结果是最有效率的
3. 成本--收益分析说明了为什么”无风险“或”零排放“政策通常是很浪费的
4. 没有管制的市场经济会产生一定水平的浪费（或其他外部性），在这个水平上，控污的私人边际收益等于私人边际成本。效率要求控污的社会边际收益等于其社会边际成本。在一个没有管制的经济中，会产生过少的控污行为和过多的污染
#### 矫正外部性问题的政策
1. 对付由外部性造成的无效率的武器是什么呢？最常见的方法是政府的反污染计划，即通过直接控制或经济激励来引导厂商矫正外部性。更细致的办法是明确并加强产权管理，以促成私人部门之间通过协商达成更加有效的解决办法
#### 政府计划
1. 直接控制。几乎所有的污染和其他影响健康与安全的外部效应，政府均依靠直接的管制加以控制；这通常被称为社会管制。理论上管制者可以选择控污法令保证经济效率，但在现实中这一点却不大可能。事实上，许多污染管制苦于越来越多的政府失灵。例如，污染管制经常在对未对边际成本和边际收益进行比较之前就排版定夺了，而没有这种比较就根本无法确定有效率的控污成本。由于使用命令--控制法，实现环保目标产生了不必要的成本
#### 市场办法：排放费
1. 为避免直接控制存在的缺陷，许多经济学家建议环境政策应更多依靠市场机制。方法之一就是收取排放费；（使得外部效应内部化）大多数环境控制利用命令--控制的办法限制个体（如能源制造厂或汽车厂）排放污染物。更重要的是，它实际上使得所有的计划都会产生无效率，因为它不满足所有排放都有相同的边际成本这个条件
#### 私人方式
1. 明确清晰的产权可以取代政府管制或税收，从而解决外部性问题
2. 私人部门克服外部性的一种方法是依靠责任规则，而不是直接的政府管制。在这种方法下，外部性问题的制造者有法律责任向受害者进行赔偿。事实上，建立一个恰当的责任系统，外部性就被内部化了
3. 责任规则从理论上讲，是将生产非市场化的成本进行内部化的一个很好的方法，但是在现实中，责任规划的应用却十分有限。它通常需要高昂的诉讼成本，这增加了原有的外部性成本，而且，由于产权却不完全（例如，涉及清新的空气时），或者外部性牵涉到大量的企业（例如，将化学物质倒入溪流中），许多受害放很难或根本无法起诉
4. 私人部门所采用的第二种方法依赖于明晰的产权和不同利益集团的谈判
5. 只要产权清晰，交易成本很低，尤其是当只有很少的利益相关方时，健全的责任规则或谈判有时也能保证在存在外部性的情况下产生有效率的结果
## 第15章 资本、利息和利润
1. 你的蛋糕完全可以既保留下来又不影响食用：将它出借并收取利息即可 
2. 本章主要研究资本。我们首先讨论资本理论的基本概念，包括”迂回性“概念和投资收益率的各种测算方法。然后我们将转向资本供求这个重要问题。这将会加深我们对自由市场经济的某些重要特征的了解
### A. 利息和资本的基本概念
#### 什么是资本
1. 资本包括那些生产出来的资本品，它们在进一步的生产中被作为生产性投入
2. 资本的基本特征是：它即使一种投入又是一种产出
3. 在早期，资本主要包括有形资产。三种重要的有形资本是：建筑（如工厂和住宅）、设备（耐用消费品，如汽车；耐用生产设备，如机床和卡车）以及投入和产出的存货（如经销商的汽车存货）
4. 现今，无形资本已经越来越重要。例如软件（如计算机操作系统）、专利（如微处理器）和品牌（如可口可乐）等。斯坦福大学的罗伯特霍尔称其为”e资本“，以区分传统的有形资本和越来越重要的知识资本
#### 投资的价格和租费
1. 大多数资本归使用它们的企业所有，但有些资本是由其所有者租借出去的。为暂时使用资本进行的支付叫做租费
2. 我们要将固定要素（如土地）的租金收入与耐用品要素（如资本）的租费区别开来
#### 资本和金融资产
1. 个人和公司都拥有各种不同种类的资产。一种就是我们上面讨论的生产投入资本。还有金融资产。金融资产本质上就是一些纸张或电子记录。更准取地说，金融资产是一方对另一方的货币要求权
2. 有形资产是经济的重要组成部分，因为它们可以提高其他要素的生产率。而金融资产的重要性产生于储蓄者和投资者之间的不匹配。学生需要钱来上大学，但他们现在没有收入或存款来支付这笔费用。而那些正在工作并为退休存钱的成年人，会有一些超过消费需求的收入可以提供给学生。金融体系机构众多，有银行、共同基金、保险公司和养老基金等，通常还有政府贷款和政府担保作为补充。通过它们可以把储蓄者的资金引向投资者
#### 投资收益率
1. 借款的成本或贷款的报酬称为投资收益率。在按固定利息计算金融资产的收益的情况下，这些收益称为利率。从经济学角度看，利率或投资收益是借款或贷款的价格
2. 资本包括生产出来的耐用品，它们在其他产品的生产中被作为生产性投入。资本包括有形资产和无形资产
3. 资本在资本市场上交易。为暂时使用资本品所做的支付叫做租费
4. 我们必须能区分金融资产，它本质上就是一些纸张，其价值源于对其他有形或无形资产的所有权
5. 投资收益率和特殊情况下的利率是借款和贷款的价格。我们通常以百分比/年为单位计算资金的收益率
#### 资本的收益率
1. 任何经济体最重要的任务之一，就是将其资本在各种可能的投资之间进行分配
2. 为了做出最佳投资决策，我们需要一种衡量资本报酬的标注。一个重要的指标是投资收益率，即每1美元资本投入在1年中能够获取的净收益
3. 投资收益率是每1美元投资的年净收益（租费收入减去费用）。它是一个纯粹的或无单位的数字--年百分比
#### 资产的现值
1. 大多数资产能在一定时期内产生一种租费流或收入流
2. 假设你对拥有并管理一幢楼感到厌倦，并已经决定出售它。为了给楼确定一个合理的价格，你需要决定该楼的全部的未来收入流的现在价值。这些收入流的价值称为该项资本资产的现值
3. 现值是未来收入流的现在的货币价值。它是这样计算的：在当前的利率下，计算出现在需要投资多少货币才能产生该项资产的未来的收入流
#### 永久性资产的现值
1. 永久性资产是一种像土地一样的资产，可以一直存在下去，每年支付N美元，直到永远。我们现在要找出现值V，如果年利率是i，那么现值就是每年收入恰为N美元的现在的投资的货币量。计算公式很简单：
```math
V = \frac{N}{i}
```
#### 折现的一般公式
1. 我们现在转入更一般的情形。对于现值的一个要点是：未来的支付没有现在的支付值钱，所以相对现在来说它们应当被折现，就像从远处的目标看起来比近处的目标要小一样。从时间的角度来看，利率也造成了类似的收缩
2. 在大多数情况下，资产的收入流量中包括好几项内容。在现值的计算中，每1美元都要按时间排好，首先，对未来收入流量的每一部分根据制度日期进行必要的贴现，计算出其现值，然后将各个现值加在一起，总额就是该项资产的现值
3. 现值V的准确计算公式如下：
```math
V = \frac{N_1}{1 + i} + \frac{N_2}{(1 + i)^2} + ... + \frac{N_t}{(1 + i)^t} + ...
```
4. 永久性资产现值是折现一般公式的特殊情况
收敛几何级数的代数公式：
```math
1 + K + K^2 + ... = \frac{1}{1 - K}
```
当 K = 1 / (1 + i) 时，可由折现一般公式得到永久性资产现值 V = N / i
#### 使现值最大化
1. 折现公式告诉我们，知道某项资产的未来收入之后，如何计算出该资产的现实价值。但是不要忘记，一项资产的未来收入通常取决于经营决策
2. 有一条规则可以给所有的投资决策一个正确的答案：计算各种可能的决策所能带来的现值，然后依照使限制最大化的原则进行权衡。用这种方法，你将会有更多的财富，随便你以任何时间任何方式花费
3. 当利率上升时，许多资产的价格就会下降
4. 资产的价格与利率呈反向变动，因为资产的现值是随着利率的上升而下降的。而且，长期资产价格的变化大于短期资产价格的变化。这是因为，大多数收益都产生于未来，因此利率的变化对长期资产价格的影响更大
5. 资产价格对利率的依赖性是金融资产的一般属性。股票、债券、不动产和许多其他长期资产的价格，都是随着利率的上升而下降的
#### 利率的神秘世界
1. 长期证券的利率通常要高于短期证券，因为只有当借款者能够增加兽医师才愿意放弃对其资金的快速存取
2. 风险较大的证券的高利率体现的是由于可能存在违约损失而需要向借款者提供补偿的数额
3. 如果一项资产在其价值只有很小减耗的情况下能够迅速转变为现金，则称该项资产具有流动性
#### 实际利率和名义利率
1. 名义利率是以1美元投资每年所得到的美元收入来衡量的。但是，美元有时会成为扭曲的标准。也就是说，以美元支付的利率不能衡量借款者实际以产品和服务形式获得的收益
2. 实际利率用以衡量我们放弃现在的商品能得到的未来的商品数量。实际利率是由名义或货币利率对通过膨胀率进行修正后得到的
3. 名义利率（有时也称为货币利率）是以货币形式衡量的货币的利率
4. 实际利率则对通货膨胀进行了修正，其计算公式为：名义利率减去通货膨胀率
### B. 资本、利润和利息理论
#### 资本的基本理论
#### 迂回性
1. 通过牺牲现在消费并生产资本品，社会可以提高其未来的消费量
#### 收益递减和对资本的需求
1. 令人惊讶地是，在过去的两个世纪，尽管我们的资本存量增长了很多倍，但资本的收益率却没有大幅下降，它一直保持在较高的水平。这是因为，创新和技术变革创造新盈利机会的速度，与旧有投资抹杀新机会的速度同样快捷。尽管计算机比许多年前功能强大了成千上万倍，但它在社会的每个角落的新用途（从医疗诊断到电子商务）都使得继续对计算机投资仍然有利可图
2. 当实际利率降到企业拥有的意愿存量与人们所愿提供的财富数量相当时，就达到了长期均衡。在长期均衡点上，净储蓄停止了，净资本积累为零，资本存量不再增加
3. 当人们愿意持有的资产价值与企业生产所需的资本量恰好相等时，在这一实际利率上，资本存量就达到了长期均衡
#### 利润作为资本的收益
1. 除工资、利息和租金之外，经济学家经常提到的第四种收入称为利润
#### 申报利润统计
1. 会计将利润定义为总收入和总成本之间的差额
2. 然而，在分析利润时，区分会计利润和经济利润很重要。会计利润（也称为营业收入）是会计在财务报表中计算的剩余收入。经济利润是减去所有成本之后的收入，包括货币成本和隐含机会成本。这些利润概念之所以不同，是因为会计利润忽略了一些隐含收益。企业所拥有的要素的机会成本称为隐含收益
3. 经济利润一般小于营业利润
#### 利润的决定因素
1. 利润实际上是各种不同收益的组合，包括所有者的资本隐含收益、承担风险的汇报好创新利润
2. 大多数所申报的企业利润，基本上是企业所有者所提供的资本和劳动等生产要素的报酬
3. 在大公司里，大多数利润都是投入的资本的机会成本
4. 利润也包括对承担投资风险的回报。所有这些风险必须要么通过购买保险来预防，要么在利润中获得风险溢价
5. 公司利润是国民收入中一个最不稳定的组成部分。因此，获得公司利润的权利（由公司股票或股权来表示）就需要一个可观的风险溢价才能吸引规避风险的投资者。这一用以抵补投资风险的股权的额外回报，称为股权溢价
6. 利润是一种剩余收入，等于总收入减去总成本。利润包括隐含收益（如所有者的资本的收益）、承担风险的回拜和创新利润
## 第16章
1. 本章考察发达工业化经济中政府的作用。市场经济中合适的经济政策目标是什么，用什么工具可以实现这些目标？一个有效率的税收体系应遵循哪些原则？把握这些问题的症结是制定有效的公共政策的关键之所在
### A. 政府对经济的控制
1. 税收政策和预算政策是社会变革的有力工具
#### 政府的政策工具
1. 政府用来影响私人经济活动的三种主要的工具。它们是：
  + 对收入、商品和服务的税收
  + 在某些商品或服务（如道路、教育或治安）领域的指出，以及为个人提供资源的转移支付（像社会保障和食品券）
  + 管制或控制措施，用以指导人们从事或减少某些经济活动。比如，对企业污染的限制，确定射频频谱标准，或是强制检验新药安全性等措施
#### 政府规模的变动趋势
1. 在每一紧急时期(萧条、战争或事关贫困和污染等严重的社会问题时)，政府的活动范围都会扩张，但危机过后，政府的控制和支出却再也没有回到原来的水平上
2. 高收入国家的税收和指出占GDP的份额比贫困国家要大
#### 政府的职能
1. 政府的四个主要职能：
  + 提高经济效率
  + 减少经济不公平
  + 通过宏观经济政策稳定经济
  + 执行国际经济政策
#### 提高经济效率
1. 政府经常运用自己的工具来矫正那些显著的市场失灵，其中最重要的是：
  + 完全竞争的破坏
  + 外部性和公共品
  + 不完全信息
#### 减少经济不公平
1. 政府保障全体公民的最低生活水平
2. 收入的再分配一般是通过税收和支出政策来进行的
#### 通过宏观经济政策稳定经济
1. 政府有义务正确地运用货币政策和财政政策，并严格管制金融体系，以制止这种灾难性的经济萧条。此外，政府还努力熨平经济的周期性波动，以防经济衰退时出现大规模的失业；经济扩张时出现严重的通货膨胀
#### 执行国际经济政策
1. 我们将国际经济政策主要分为四个方面：
  + 减少贸易壁垒
  + 执行援助计划
  + 协调宏观经济政策：各国都已发现，其他国家的财政政策和货币政策会影响本国的通货膨胀、失业和金融形势。国际货币体系不可能依靠自身来运转，稳定运行的汇率体系是有效率的国际贸易体系的先决条件之一。一国的通货膨胀、失业或金融危机就不在外溢，从而避免损害整个地区
  + 保护全球环境
#### 公共选择理论
1. 政府往往也会做出错误的决定，获奖很好的计划执行得非常糟糕
2. 这些问题属于公共选择理论范畴。这种理论是一种研究政府决策方式的经济学和政治学。公共选择理论考察不同选举机制的运作方式，指出没有一种理想的机制能够将所有的个人偏好综合为一种社会选择；研究当国家干预不能提高经济效率或收入在分配存在不公平时所产生的政府失灵；还研究国会议员的短视，严格预算约束的缺乏，以及为竞选提供资金所导致的政府失灵等问题
### B. 政府支出
#### 文化和技术影响
1. 除了支出金钱，政府计划还能对全国产生许多微妙的影响
2. 互联网就是因国防部为建立一个在核战争中能够继续有效运行的通信网络逐步发展而成的
### C. 税收的经济学问题
1. 税是我们为文明社会所支付的代价--奥利佛温德尔霍姆斯大法官
2. 在税收所造成的货币流动的背后，政府实际需要的是经济学中稀缺的土地、劳动和资本
3. 通过税收，政府实际上在决定如何从公民和企业的手中取得资源以用于公共目标。通过税收所筹集的货币实际上只是一种载体，经由它才能将那些现实的经济资源由私人品转化为公共品
#### 税收原则
1. 指定税收制度的两个主要原则：
  + 受益原则认为，个人所缴纳的税额英语他们从政府计划中得到的利益成比例
  + 支付能力原则认为，人们缴纳税的数额应与其收入或财富相对应
2. 例如，若建造一座新桥的资金要靠过桥费莱偿付，则这里所体现的就是受益原则。因为只有你使用它时才支付费用。但若建桥资金来自于所得税，则它所依据的就是支付能力原则
#### 横向公平和纵向公平
1. 横向公平。这个原则指出，那些在实质上没有差别的人在那谁上也应该没有差别
2. 纵向公平。它是关于如何确定不同收入水平的人们的纳税待遇的
3. 横向公平指的是每个人都有权得到平等待遇的原则。而纵向公平指的是，地位不同的人有权公平地享有不尽相同的待遇。不过，关于纵向公平到底应该如何应用，始终没有达成共识
#### 税收中的实用主义折中
1. 受益税在政府收入中的份额正在下降。现在，发达国家更多地依赖于累进所得税。在累进税下，一个收入5万美元的家庭的纳税额要高于一个收入2万美元的家庭。高收入家庭不仅要交更多的所得税，而且其纳税占收入的份额实际上也更高
2. 累进税：随着收入的增加，税收在收入中所占比例增加
3. 累进税不同于严格的比例税。比例税要求所有纳税人按其收入的相同比例进行纳税
4. 累退税则使低收入家庭的纳税占收入的份额要高于高收入家庭
5. 税收可被划分为比例的、累进的还是累退的，取决于高收入者所纳税占收入的比例是等于、大于还是小于低收入者所纳税额占其收入的比例
6. 经济学家还发现，香烟税是累退的。因为香烟购买量上升的速度要比收入上升的速度慢
7. 间接税是对商品和服务征收的，所以是”间接地“对个人征收。例如货物税和销售税，烟草税和汽油税，进口酷灌水以及财产税等。而直接税是对个人或企业征收的。例如个人所得税、社会保险或其他工薪税，以及遗产税和赠与税等。直接税的优点是更容易进行调整，以适应不同人的不同情况，如家庭规模、收入、年龄和一般支付能力。与此相比，间接税的优点是征收成本比较低，征管上也比较方便，因为它们可以在零售和批发环节上征收
#### 个人所得税
1. 有效税率或平均税率等于总纳税额除以总收入
2. 边际税率是因为多增加1美元收入而需要多缴纳的税率
3. 在税收分析中边际税率是个很关键的指标，因为个人和公司都总是倾向于注重他们的边际税率而不是平均税率。进一步说，当边际税率特别高的时候，人们的积极性将会严重受挫
4. 记住”边际原则“：人们应该关心的是所发生的新增成本或新增收益；而”过去的就（应该）让它过去“。就此而言，任何税收对吉利的影响都主要来自于边际税率
#### 社会保险税
1. 雇员根据其收入记录和过去的社会保险缴纳情况得到退休金
#### 消费税
1. 销售税和货物税通常都是累退的，因为这些物品的消费者在贫困家庭的收入中所占的比例要大于在富裕家庭中所占的比例
2. 一种在美国之外的其他国家被普遍采用的是增值税（VAT）。增值税很像销售税，但它在生产的每一阶段都进行课征
#### 财产税
1. 财产税主要对土地和建筑等不动产进行征收
#### 税收的效率与公平
#### 有效征税的目标
1. 对劳动所得的课征。如果劳动供给曲线是向后弯曲的，则会产生劳动随税率下降而减少的悖论
2. 对资本所得的课征。当一个部门的税收很高时，资源就会流向征税较轻的部门
3. 全球化的影响。一国需要保证本国生产要素如资本和高技术劳工不会被吸引和流动到地税收国家
#### 效率和公平
1. 亨利乔治曾经说过：对土地征税对效率不会有什么影响，因为土地的供给是完全无弹性的。有效税收的现代理论提出了拉姆齐税收原则。她人为政府应对那些供给和需求最没有价格弹性的投入和产出征收最终的税。拉姆齐税收原则的理论基础是：如果一种产品在供给或需求上非常缺乏价格弹性，那么对这种产品征税对其消费和生产的影响就十分有限。在某些情况下，拉姆齐税收可以说是以最小的经济效率损失增进收入的一种办法
## 第17章 效率与公平：重大权衡
1. 本章旨在考察收入分配和减少不公平的两难政策抉择
### A. 不公平的根源
1. 要衡量人们控制经济资源的不公平，我们需要弄清收入与财富的区别。请记住，个人收入这一概念指的是某人或某家庭在某个时期（通常是一年）所获得的总的款项或现金。个人可支配收入指的是个人收入减去所有的赋税。财富或”净价值“则是某个时点上所拥有的有形资产和金融资产的货币价值，减去对银行和其他信用机构的欠款
#### 如何衡量个收入阶层之间的不公平
1. 洛伦兹曲线被广泛地用于对收入和财富不公平状况的分析
2. 基尼系数。经济学家常常需要将不公平的程度进行量化。常用的量化指标是基尼系数。通过计算洛伦兹曲线阴影部分的面积乘以2可以得出基尼系数。在绝对不公平下，基尼系数等于1，而在绝对公平时基尼系数等于0
#### 财富的分布
1. 财富指的是金融资产和有形资产的完全所有权
#### 美国的贫困问题
1. 将贫困的概念变为相对收入。专家组建议当某家庭的消费还不到平均家庭在视频、服装、住房上的消费水平的50%时，该家庭即可被定义为贫困
#### 谁是富人
1. 许多最富的人的高收入主要来源于财产性收入，由股票、债券和不动产所带来的收入所组成
### B. 反贫困政策
#### 福利国家的兴起
1. 这就是福利国家兴起的标志。在福利国家中，政府调节市场力量以保护个人能应付某些偶然事件，并保证人民有最起码的生活水平
2. 福利国家的政策主要包括公共养老金、意外伤残保障、失业保险、医疗保险、食品和住房计划、家庭补偿以及对某些群体的收入补助
#### 再分配的代价
1. 公平有哪些不同的概念？首先，民主社会注重政治权利的平等
；人们应享有公平的经济机会；经济结果的公平（”各尽所能，按需分配“）
2. 一味地坚持结果的公平会严重阻碍经济机制发挥作用
#### 漏洞有多大
1. 奥肯用一只桶来刻画我们的税收和转移支付再分配体系的特征。由于高税率和慷慨的收入支持计划造成的主要无效率：管理成本的增加、工作和储蓄积极性的挫伤，以及社会经济为此付出的代价。
#### 漏泄的加总
1. 国家需要仔细设计其政策，以避免不可接受的不公平或重大效率损失等极端状况
## 第18章 国际贸易
### A. 国际贸易的实质
#### 国际贸易与国内贸易
1. 国内贸易和国际贸易之间还是存在三大差异:
  + 扩展了的贸易机会
  + 主权国家：国家有时会通过关税或配额，对国际贸易设置壁垒，以”保护“会受影响的工人或企业，是其免遭国外竞争的冲击
  + 国际金融
#### 美国对外贸易发展趋势
1. 在一个特定的行业中，美国在同一时期内往往既需要进口也需要出口，因为产品的高差异性已经导致不同国家都可以在该产品市场上占有一席之地
#### 商品和服务的国际贸易原因
1. 国际贸易模式背后的经济因素是什么？基于以下三种原因，各国都发现参与国际贸易是有利可图的：生产条件的多样新，各国偏好不同，生产成本随规模递减
### B. 国家间的比较优势
#### 比较优势原则
1. 比较原则认为，即使一国在各种商品的生产上都比其他国家都绝对地更有效率（或绝对地更缺少效率），该国也仍然能够从国际贸易中获益
#### 似乎有悖常识
1. 比较优势原则认为，如果各国专门生产和出口其生产成本相对低的产品（和自己比较），就会从贸易中获益。或者反过来说，如果各国进口其生产成本相对高的产品，也将从贸易中得利
#### 贸易的经济利益
1. 在自由贸易条件下，当各国集中在自己具有比较优势的领域进行生产和贸易时，每个国家的情况都会变得比原先要好。与没有贸易的情况相比，当各国的劳工专门生产自己具有比较优势的产品，并用以交换比较劣势的产品时，他们工作同样的时间就能获得更多的消费品
#### 外包作为另一种贸易形式
1. 外包是指将服务业务或生产加工等分派到海外去进行
2. 外包贸易可视为比较优势原则在更多领域的推广
#### 开放市场进行贸易
1. 我们将出口价格与进口价格的比率称为贸易条件
2. 具体的比率取决于需求的力量
3. 竞争市场中的自由贸易使得世界生产能够上移到其生产可能性边界上
#### 扩展到多种商品和多个国家
#### 限制条件和结论
1. 现在我们已经比较系统地学习了比较优势理论，它的结论适用于任何数量的国家和商品，而且可以被一般化以解决多种投入、要素比例变化及收益递减等问题。但我们必须强调这个理论有两个重要的限制条件：
  + 古典假设。当错在本地或全球公共品的时候，贸易可能导致环境问题的恶化。而且，非弹性价格和工资、商业周期以及非自愿事业可能会导致低效率
  + 收入分配
2. 斯托珀萨缪尔森定理表明：已过开放贸易会增加国民收入，能消费到比不进行贸易时更多的商品和服务。这并不意味着每个个人、企业、部门或生产要素都能从贸易中获益
3. 尽管比较优势理论有其局限性，但他是经济学中最深刻的真理之一。那些忽视比较优势的国家，在生活水平和经济增长方面，会因策而付出沉重的代价
### C. 贸易保护主义
1. 在自由贸易下，或更一般地说在市场上，商品总是从价格低的地区流向价格高的地区
#### 贸易壁垒
1. 关税是对进口商品课征的一种税
2. 配额是一种对进口商品的数量限制
3. 关税会提高商品的价格，减少消费和进口的数量，增加国内生产
4. 配额与关税的影响在性质上是相同的，一个禁止性配额（阻止所有进口的配额）相当于一项禁止性关税
5. 尽管关税和配额之间没有本质性的差别，但仍存在一些微妙的区别。关税能给政府带来收入
6. 经济学家一般认为关税的危害相比之下要要小一些
#### 关税的经济成本
1. 关税会造成经济无效率。征收关税时，消费者的经济损失超过政府的收入与厂商所得的额外利润的总和
2. 征收关税时，部分经济影响是由关税对收入的再分配造成的，它将一部分收入从消费者哪里转移到被保护的国内生产者和劳工手中
3. 征收关税会带来三种影响：它鼓励了无效率的国内生产；它提高了价格，从而导致消费者对征税商品的购买量低于有效率的水平；它增加了政府收入。其中前两项都给经济带来了效率成本
#### 保护主义经济学
1. 要求以关税或配额保护本国不受外国进口品竞争损害的主张，往往表现为多种形式，以下是主要的几类：（1）非经济观点，这种观点认为牺牲一些经济福利以支持国家其他目标是必要的；（2）若干误解经济逻辑的观点；（3） 基于市场力量或宏观经济的不完善而进行的若干分析
#### 站不住脚的关税观
1. 货币的价值不在于它本身，而在于它能从其他国家买到些什么
#### 有潜在正确性的贸易保护主义
1. 最后，我们评价三个有实际经济学价值的贸易保护主义论点：
  + 关税可能使贸易条件变得对一国有利
  + 对于有增长潜力的”幼稚产业“实行临时性关税保护，从长期看可能是有效率的
  + 在某些条件下，关税有助于减少失业
2. 能够最大化国内实际收入的关税称为最优关税
3. 贸易条件理论可以追溯到150年前的自由贸易鼓吹者约翰斯图亚特穆勒那里。它是在充分就业和完全竞争条件下唯一可能正确的关税理论。假设美国对进口石油按最优关税征税，则导致国内石油价格上升，世界对是有的需求下降，世界市场的石油价格也将下降。因此灌水的一部分实际上会落到石油生产者的身上。即”损人利己“。（可以推知小国就不能运用这个理论，因为它不能影响世界价格）
4. 历史上，实行贸易保护主义的一个强有力的动机是，这样做在经济衰退或滞胀时期有助于增加就业机会
5. 对于增加就业机会和降低失业来说，关税和进口保护都不是有效率的做法。一种更有效的增加就业的途径是运用国内的货币政策和财政政策
#### 其他贸易壁垒
1. 我们因该注意哪些所谓的非关税壁垒（NTB）。这些非正式的限制和管制使得一国很难在外国市场上出售自己的商品
2. 近些年，非关税壁垒的保护程度约为关税壁垒的两倍
#### 多边贸易谈判
1. 在关税报复游戏的结局中，所有的人都是输家
#### 评价
1. 经济学研究表明，各国从更加开放的贸易中得到了利益，因为它带来了贸易量的增加和更高的生活水平。但是，因为政策和经济环境不断变化，保护开放市场所进行的努力常常要接受各种检验
#### 总结提要
1. 如果由于失业和外汇市场的扰动，市场不能正常运行，那么比较优势原则就要加以修正。而且，如果进口品降低了某些部门和要素的收益，那么这些部门或要素就会因贸易而受到损害
